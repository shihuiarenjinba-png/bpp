## 基本的なメトリクス

### 投資金額

**定義:** 各銘柄の購入時の総投資額

**計算式:**
```
投資金額 = 取得価格 × 数量
```

**例:**
- 取得価格: ¥2,100, 数量: 100株
- 投資金額 = ¥2,100 × 100 = ¥210,000

**用途:** ポートフォリオ全体の規模把握、配分割合の計算

---

### 現在評価額

**定義:** 各銘柄の現在時点の総評価額

**計算式:**
```
現在評価額 = 現在価格 × 数量
```

**例:**
- 現在価格: ¥2,450, 数量: 100株
- 現在評価額 = ¥2,450 × 100 = ¥245,000

**用途:** ポートフォリオの現在価値把握、実現損益の計算

---

### 利益（取得時からの利益）

**定義:** 購入から現在までの未実現利益

**計算式:**
```
利益 = 現在評価額 - 投資金額
     = (現在価格 - 取得価格) × 数量
```

**例:**
- 現在評価額: ¥245,000, 投資金額: ¥210,000
- 利益 = ¥245,000 - ¥210,000 = ¥35,000

**注意:** 実現していない利益なため、売却時のコストや税金は考慮されていません

---

## リターン関連メトリクス

### リターン率

**定義:** 投資金額に対する価格変動の割合

**計算式:**
```
リターン率 = (現在価格 - 取得価格) / 取得価格
          = 利益 / 投資金額
```

**例:**
- 取得価格: ¥2,100, 現在価格: ¥2,450
- リターン率 = (¥2,450 - ¥2,100) / ¥2,100 = 16.67%

**特徴:**
- 価格感度に関わらず、投資効率を統一的に比較できる
- 通常、パーセンテージで表示される

---

### ウェイト（配分比率）

**定義:** ポートフォリオ全体に占める各銘柄の投資金額の割合

**計算式:**
```
ウェイト = 投資金額 / 総投資金額 × 100%
```

**例:**
- 銘柄A投資金額: ¥210,000, 総投資金額: ¥1,000,000
- ウェイト = ¥210,000 / ¥1,000,000 = 21.0%

**用途:**
- ポートフォリオの構成確認
- リスク配分の可視化
- 集中リスクの評価

---

### ポートフォリオリターン（加重平均リターン）

**定義:** ポートフォリオ全体の平均的なリターン率

**計算式:**
```
ポートフォリオリターン = Σ (ウェイト × リターン率)
```

**例:**
```
銘柄A: ウェイト 40%, リターン率 10%  → 加重リターン 4.0%
銘柄B: ウェイト 30%, リターン率 20%  → 加重リターン 6.0%
銘柄C: ウェイト 30%, リターン率 -5%  → 加重リターン -1.5%

ポートフォリオリターン = 4.0% + 6.0% - 1.5% = 8.5%
```

**特徴:**
- ポートフォリオ全体の期待収益率を示す
- 個別銘柄のリターンより安定的（分散効果）

## リスク関連メトリクス

### 標準偏差（ポートフォリオ標準偏差）

**定義:** ポートフォリオのリターン変動性（ボラティリティ）を示すリスク指標

**計算式:**
```
標準偏差 = √[Σ(加重リターン - 平均リターン)² / n]
```

**例:**
```
加重リターン: [4%, 6%, -1.5%, 3%, 5%]
平均: 3.4%
標準偏差 = √[(0.6² + 2.6² + (-4.9²) + (-0.4)² + 1.6²) / 5]
        = √[4.84 / 5] = 0.986 ≈ 0.99 (99bps)
```

**特徴:**
- 値が大きいほどリターンが変動しやすい（リスク大）
- 値が小さいほど安定している（リスク小）
- 標準偏差が同じなら、シャープレシオで比較する

**解釈目安:**
- 0.5未満: 低リスク（社債に近い）
- 0.5～1.5: 中リスク（バランス型投資に相当）
- 1.5以上: 高リスク（成長株中心）

---

### 分散

**定義:** 標準偏差の二乗、リスク度のもう一つの表現

**計算式:**
```
分散 = 標準偏差²
```

**特徴:**
- 統計学的な計算には分散を使用
- 解釈としては標準偏差の方が直感的

---

### 最大ドローダウン（Maximum Drawdown）

**定義:** ポートフォリオが最高値から最低値に下がった最大の下落率

**計算式:**
```
最大ドローダウン = (最低値 - ピーク時の値) / ピーク時の値
                = (最低評価額 - 最高評価額) / 最高評価額
```

**シミュレーション例:**
```
時点1: 評価額 ¥100万 (ピーク)
時点2: 評価額 ¥95万
時点3: 評価額 ¥80万 (最低)
時点4: 評価額 ¥90万

最大ドローダウン = (¥80万 - ¥100万) / ¥100万 = -20%
```

**特徴:**
- 負の値で表示される
- 最悪シナリオでの損失度を示す
- リスク許容度の低い投資家にとって重要

**解釈目安:**
- -10%以上: 低リスク（損失が小さい）
- -10%～-25%: 中程度（バランス型相当）
- -25%以下: 高リスク（大きな損失の可能性）

## リスク調整後リターンメトリクス

これらのメトリクスは、同じリスクレベルに対するリターンの効率性を評価します。

### シャープレシオ（Sharpe Ratio）

**定義:** リスク1単位あたりの超過リターン（リスクフリーレート超過分）

**計算式:**
```
シャープレシオ = (ポートフォリオリターン - リスクフリーレート) / 標準偏差
```

**例:**
```
ポートフォリオリターン: 8.5%
リスクフリーレート: 2.0% (日本国債利回り等)
標準偏差: 0.99 (0.99%)

シャープレシオ = (8.5% - 2.0%) / 0.99% = 6.57
```

**特徴:**
- リスク1単位あたりいくらのリターンを得ているかを示す
- ポートフォリオ間の効率性を直接比較可能
- リスクフリーレート選択が結果に影響

**解釈目安:**
- 0.5未満: 効率が悪い
- 0.5～1.0: 平均的（S&P500 ≈ 0.6）
- 1.0～2.0: 良好
- 2.0以上: 優秀
- 負値: リターンがリスクフリーレート以下

**注意点:**
- リスクフリーレート（rf）選択に注意
  - 日本国債10年: 2026年4月時点で約0.5～1.0%
  - 米国財務省10年: 約4.0～5.0%

---

### ソルティーノレシオ（Sortino Ratio）

**定義:** シャープレシオの改良版。下方向のリスク（損失側の変動）のみを考慮

**計算式:**
```
ソルティーノレシオ = (ポートフォリオリターン - リスクフリーレート) / 下方偏差

下方偏差 = √[Σ min(リターン - 目標リターン, 0)² / n]
```

**例：**
```
ポートフォリオリターン: 8.5%
リスクフリーレート: 2.0%
下方偏差: 0.5% (負のリターンのみを考慮)
標準偏差: 0.99%

シャープレシオ = (8.5% - 2.0%) / 0.99% = 6.57
ソルティーノレシオ = (8.5% - 2.0%) / 0.5% = 13.0
```

**特徴:**
- 上昇のボラティリティは無視する
- 投資家は上昇の変動は許容するため、より実用的
- シャープレシオより大きな値になる傾向

**解釈目安：**
- 2.0未満: 低い
- 2.0～3.0: 良好
- 3.0～5.0: 優秀
- 5.0以上: 非常に優秀

## 相関性と分散効果

### 相関係数（Correlation Coefficient）

**定義:** 2つの銘柄のリターンの連動性（-1～+1）

**計算式:**
```
相関係数 = Σ[(リターンA - 平均A) × (リターンB - 平均B)] / (n × 標準偏差A × 標準偏差B)
```

**例：**
```
銘柄A（自動車）と銘柄B（電力）のリターン推移:

銘柄A: 10%, 15%, 5%, 8%, 12%
銘柄B: 2%, 3%, 1%, 4%, 2%

相関係数 = 0.85（強い正の相関）
```

**解釈:**
- **+1.0に近い**: 2銘柄が同じ方向に動く（分散効果なし）
- **0に近い**: 2銘柄の動きに関連性なし（分散効果あり）
- **-1.0に近い**: 2銘柄が反対方向に動く（最高の分散効果）

**ビジネス例:**
```
相関係数 0.9以上: 
- 自動車メーカー複数社
- 同業種の企業
→ 分散効果が薄い

相関係数 0.0～0.3:
- 異なる業種（自動車 × 医薬品）
→ 分散効果がある

相関係数 -0.3以下:
- 不動産 × エネルギー（時期により反対動向）
→ 高い分散効果
```

**ハーフィンダール指数との関係:**
- 高い相関係数のポートフォリオは、ウェイトが均等でもリスク軽減効果が薄い
- 低い相関係数のポートフォリオは、分散投資の効果が高い

## セクター分析メトリクス

### セクター構成比（Sector Allocation）

**定義:** ポートフォリオ全体のうち、各業種が占める割合

**計算式:**
```
セクター構成比 = セクター総投資金額 / 総投資金額
```

**例：**
```
総投資金額: ¥1,000,000

セクター別投資金額:
- 自動車: ¥400,000 → 構成比 40%
- 情報通信: ¥300,000 → 構成比 30%
- 医薬品: ¥200,000 → 構成比 20%
- 電力: ¥100,000 → 構成比 10%
```

**用途:**
- 景気サイクルに基づくセクター配置の妥当性評価
- 業種リスク（セクターリスク）の把握
- 経済変化への耐性確認

---

### セクターリターン（Sector Return）

**定義:** 各セクター内の銘柄リターンの加重平均

**計算式:**
```
セクターリターン = (セクター現在評価額 - セクター投資金額) / セクター投資金額
```

**用途:**
- 業種別のパフォーマンス評価
- 強気・弱気セクターの特定

---

### ハーフィンダール指数（Herfindahl Index）

**定義:** ポートフォリオの集中度を測定するリスク指標

**計算式:**
```
ハーフィンダール指数 = Σ(セクター構成比)²
```

**例：**
```
シナリオ1（均等配分）:
セクターA: 25%, B: 25%, C: 25%, D: 25%
HI = 0.25² + 0.25² + 0.25² + 0.25² = 0.25

シナリオ2（集中配分）:
セクターA: 50%, B: 30%, C: 15%, D: 5%
HI = 0.50² + 0.30² + 0.15² + 0.05² = 0.365
```

**範囲と解釈：**

| ハーフィンダール指数 | 集中度評価 | リスクレベル |
|------------------|--------|---------|
| 0.10～0.15 | 低 | 十分に分散 |
| 0.15～0.25 | 中 | 標準的 |
| 0.25～0.50 | 高 | 集中リスク有り |
| 0.50以上 | 極度 | 極度に集中 |

**解釈:**
- 値が小さいほど分散が進んでいる
- 値が大きいほど特定セクターに偏っている
- リスク許容度によって目標値を設定

**最小値:** 1/n（完全に均等配分）
- 2セクター: 最小0.50
- 4セクター: 最小0.25
- 10セクター: 最小0.10

## 解釈ガイド

### ポートフォリオ評価の総合判断

#### 1. 効率性の評価（シャープレシオ）

```
シャープレシオ 0.5以下
→ 改善の余地あり。リターンに対してリスクが大きい
    対策: 低相関の銘柄追加、ウェイト調整

シャープレシオ 0.5～1.0
→ 平均的。S&P500並みの効率性
    対策: 特に問題なし。銘柄見直し時に検討

シャープレシオ 1.0～2.0
→ 優秀。リスク調整後リターンが良好
    対策: 現在のバランスを維持

シャープレシオ 2.0以上
→ 非常に優秀。持続可能か検証が必要
    対策: 過去データで検証、サンプル期間バイアスに注意
```

---

#### 2. リスク水準の評価（標準偏差 × 最大ドローダウン）

```
標準偏差 0.5%, 最大ドローダウン -5%
→ 低リスク（債券・安定型ファンド相当）

標準偏差 1.0%, 最大ドローダウン -15%
→ 中リスク（バランス型投資信託相当）

標準偏差 2.0%, 最大ドローダウン -30%
→ 高リスク（成長株・インデックスファンド相当）

標準偏差 3.0%+, 最大ドローダウン -50%+
→ 極度に高リスク（投機的、初心者向けでない）
```

---

#### 3. 分散効果の評価（相関係数 × ハーフィンダール指数）

```
ケース1: 高相関 (0.7+), 高集中 (HI > 0.25)
→ 分散効果が極めて薄い。即座に改善が必要
   同業種が多すぎる、または同じテーマ銘柄

ケース2: 高相関 (0.7+), 低集中 (HI < 0.15)
→ 銘柄数を増やしても分散効果が限定的
   相関の低い異業種・異資産クラスを導入

ケース3: 低相関 (0.3-), 高集中 (HI > 0.25)
→ 特定銘柄のウェイトが大きい。段階的に調整

ケース4: 低相関 (0.3-), 低集中 (HI < 0.15)
→ 理想的なポートフォリオ。現状を維持
```

---

### 実践的なシナリオ分析

#### シナリオA: 高リターンだが高リスク

```
ポートフォリオリターン: 15%
標準偏差: 2.5%
シャープレシオ: 5.2
最大ドローダウン: -35%
```

**評価:** リターンは優秀だが、最大ドローダウンが大きい
**改善案:**
1. 相関の低い防衛銘柄（医薬品、生活必需品）を追加
2. ハイリスク銘柄のウェイトを段階的に低減
3. ソルティーノレシオを計算し、損失側のリスクを評価

---

#### シナリオB: リターン低迷

```
ポートフォリオリターン: 2.5%
標準偏差: 0.8%
シャープレシオ: 0.06
最大ドローダウン: -8%
```

**評価:** リスクは低いがリターンが不十分、リスク調整後のリターンが悪い
**改善案:**
1. ポートフォリオ構成の再検討（値上がり期待銘柄の比率を増加）
2. 業績良好セクターへの配置換え
3. 現在のリスク許容度を再評価

---

#### シナリオC: バランス型（理想的）

```
ポートフォリオリターン: 8%
標準偏差: 1.2%
シャープレシオ: 0.5
最大ドローダウン: -12%
相関係数平均: 0.35
ハーフィンダール指数: 0.18
```

**評価:** リスク・リターンのバランスが取れている
**対策:** 現在のバランスを維持しつつ、四半期ごとにリバランス

### よくある質問と回答

**Q: シャープレシオはいくつ以上が「良い」のか？**

A: 相対的な評価が重要です。
- 同じリスク水準の他のポートフォリオとの比較
- 自分のリスク許容度との比較
- 市場全体（S&P500など）との比較（0.6程度が目安）

日本株中心なら0.3～0.5が平均的です。

---

**Q: 相関係数が0.8と高い場合、どうすればよい？**

A:
1. **セクター多様化**: 金融 × 不動産の相関は高い傾向。異なる景気サイクル銘柄を追加
2. **国・地域の多様化**: 日本株のみなら海外株を追加
3. **資産クラスの追加**: 株式のみなら債券やREITを組み込み

---

**Q: 最大ドローダウン -30% は許容できるか？**

A: 個人の投資期間と心理面による：
- 短期投資家 (< 3年): -10%以内が目安
- 中期投資家 (3-10年): -20%程度が許容可能
- 長期投資家 (10年+): -30%でも耐えられる可能性

重要なのは「自分が実際に耐えられるか」を試す（心理テスト）ことです。

---

**Q: ハーフィンダール指数 0.22 は良いのか悪いのか？**

A: 「中程度の集中」です。
- 4セクター均等 (0.25) に比べてやや分散
- 10セクター均等 (0.10) に比べて集中

銘柄数が少ない（5社未満）なら許容可能。
10社以上なら、さらに分散を進めることを検討してください。
