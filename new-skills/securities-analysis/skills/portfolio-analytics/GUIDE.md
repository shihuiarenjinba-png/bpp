# ポートフォリオ分析スキル

投資ポートフォリオの包括的なリスク・リターン分析を実施するスキルです。複数の銘柄を保有する場合、その全体的なパフォーマンスと分散効果を可視化します。

## 利用方法

### ステップ1: CSVファイルの準備

以下の形式でCSVファイルを作成してください。文字コードはUTF-8で統一してください。

```
銘柄,取得価格,数量,現在価格,セクター
トヨタ自動車,2100,100,2450,自動車
ソフトバンク,1200,50,1850,情報通信
NTT,2800,30,3100,通信
東京電力,1100,200,1350,電力
武田薬品,3200,40,3500,医薬品
```

**カラムの説明:**
- **銘柄**: 株式銘柄の名称（任意の文字列）
- **取得価格**: 購入時の価格（円、数値のみ）
- **数量**: 保有株数（数値のみ）
- **現在価格**: 現在の株価（円、数値のみ）
- **セクター**: 業種区分（例：自動車、情報通信、医薬品など）

### ステップ2: 分析の実行

以下のコマンドを実行します。

```bash
python scripts/portfolio_analyzer.py --input ポートフォリオデータ.csv --output ポートフォリオ分析レポート.xlsx
```

オプション引数:
- `--rf-rate`: リスクフリーレート（年率、デフォルト値: 0.02、例：0.03で3%）
- `--periods`: 年間営業日数（デフォルト値: 252日）

### ステップ3: 出力レポートの確認

生成されたExcelファイルには以下のシートが含まれています。

#### サマリーシート
- ポートフォリオ全体のサマリー統計
- 総投資金額、現在の評価額
- ポートフォリオリターン（率・額）
- リスク指標一覧

#### 個別銘柄分析シート
- 各銘柄の詳細情報
- 投資金額、現在評価額、リターン
- ポートフォリオ全体に占める割合
- 各銘柄のリスク度（標準偏差）

#### リスク指標シート
- シャープレシオ
- ソルティーノレシオ
- 最大ドローダウン
- 相関係数行列

#### 相関行列シート
- 各銘柄間の相関係数
- ヒートマップで可視化

#### セクター配分シート
- セクター別の投資額と構成比
- セクター集中リスク（ハーフィンダール指数）

## 分析指標の説明

詳細な計算式と解釈方法については、`references/portfolio_metrics.md`をご参照ください。

## チャートの見方

### 配分比率パイチャート
ポートフォリオの銘柄別構成比を示します。大きなリスク偏在がないか確認できます。

### リターン棒グラフ
各銘柄の個別リターンを比較します。どの銘柄が収益に貢献しているかが一目瞭然です。

### 相関係数ヒートマップ
銘柄間の相関関係を色で表示します。相関が低い（色が薄い）ほど分散効果があります。

## 注意事項

- **価格データの信頼性**: 分析結果は入力データの正確性に大きく依存します。取得価格と現在価格は正確に入力してください
- **サンプル期間**: 過去データを用いた分析ではなく、現在の構成に基づいた分析です
- **リスク指標の解釈**: シャープレシオなどのリスク調整後リターンは、過去のパフォーマンスに基づくものであり、将来のパフォーマンスを保証するものではありません
- **セクター分類**: セクター名は任意ですが、分析の有効性のため、各銘柄を適切に分類してください

## トラブルシューティング

**エラー: ファイルが見つかりません**
- ファイルパスと名前を確認してください
- ファイルはスクリプトと同じディレクトリか、絶対パスで指定してください

**エラー: 不正な列が見つかりました**
- CSVの列名が正確に「銘柄,取得価格,数量,現在価格,セクター」であることを確認してください
- 文字コードをUTF-8に統一してください

**エラー: 数値解析に失敗**
- 価格と数量のセルに数値のみが入力されていることを確認してください
- 余分なスペースや記号がないか確認してください
