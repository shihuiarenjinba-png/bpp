# 配当割引モデル理論と参考資料

## 概要

このドキュメントは、このスキルで使用される配当割引モデル（Dividend Discount Model, DDM）の理論的背景を説明します。

---

## 1. ゴードン成長モデル (Gordon Growth Model)

### 概要

ゴードン成長モデルは、永続的に一定の成長率で成長する配当に基づいて、企業の理論的価値を計算する単純で強力なモデルです。

### 公式

```
P₀ = D₁ / (r - g)

ここで：
P₀  = 現在の理論株価
D₁  = 次年度の予想配当
r   = 要求利回り（期待リターン）
g   = 永続成長率（配当成長の永続的な見通し）
```

### 各要素の説明

#### **D₁（次年度予想配当）**

最直近年度の実績配当に、今後の成長率を適用して計算します。

```
D₁ = D₀ × (1 + g)

D₀: 最直近実績配当
g: 成長率
```

例：
- 2023年実績配当: 100円
- 予想成長率: 5%
- D₁ = 100 × 1.05 = 105円

#### **r（要求利回り）**

投資家が望む最低限のリターン率です。企業のリスクが高いほど高くなります。

計算方法（CAPM）：
```
r = rf + β(rm - rf)

rf   = リスク・フリー・レート（国債利回りなど）
β    = ベータ値（企業固有のリスク指標）
rm   = 市場平均リターン
rm-rf = リスク・プレミアム
```

一般的な値：
- 低リスク企業: 5-6%
- 中リスク企業: 7-9%
- 高リスク企業: 10-12%

#### **g（永続成長率）**

配当が永遠に成長し続けると仮定した成長率です。一般的に以下の範囲：

```
g = ROE × 内部留保率

ROE = 株主資本利益率
内部留保率 = 1 - 配当性向
```

制限：
- g < r である必要があります（理論的に）
- 実務上は g ≤ 3% が合理的です（GDP成長率程度）

### 例

#### シナリオ

- 最新年配当: 200円
- 成長率: 4%
- 要求利回り（r=7%）

#### 計算

```
D₁ = 200 × 1.04 = 208円
P₀ = 208 / (0.07 - 0.04) = 208 / 0.03 = 6,933円
```

現在株価が5,000円なら、理論値より安値で、割安と判断されます。

### GGMの利点

✓ シンプルで計算が容易
✓ 安定配当企業の評価に適している
✓ 直感的に理解しやすい

### GGMの限界

✗ 永遠に一定成長を仮定（現実的でない）
✗ 成長率の小さな変化で結果が大きく変わる（感度が高い）
✗ 成長段階の企業や無配当企業に適用不可
✗ r > g の条件が外れると使用不可

## 2. 多段階配当割引モデル (Multi-Stage DDM)

### 概要

多段階DDMは、企業のライフサイクルを複数の成長段階に分け、各段階で異なる成長率を適用します。

### 2段階モデル

```
P₀ = Σ[t=1 to n](D₀ × (1 + gₕ)ᵗ / (1 + r)ᵗ) + (Dₙ × (1 + gₗ) / (r - gₗ)) / (1 + r)ⁿ

ここで：
gₕ = 高成長期の成長率（高成長段階）
gₗ = 低成長期の成長率（安定段階）
n  = 高成長期間の年数
```

### 実務的な2段階モデル例

企業を2つの段階に分類：

| 段階 | 期間 | 特徴 | 成長率の例 |
|------|------|------|-----------|
| 高成長期 | 5-10年 | 市場拡大、事業成長が急速 | 10-20% |
| 安定成長期 | 11年目以降 | 市場成熟、GDP成長率並み | 2-4% |

### 例

#### 前提
- 2023年実績配当: 100円
- 高成長期（5年）: 15%成長
- 安定成長期: 3%成長
- 要求利回り: 8%

#### 計算

**第1段階（高成長期5年の現価）**
```
年1: 100 × 1.15 / 1.08 = 106.48
年2: 100 × 1.15² / 1.08² = 98.60
年3: 100 × 1.15³ / 1.08³ = 91.41
年4: 100 × 1.15⁴ / 1.08⁴ = 84.68
年5: 100 × 1.15⁵ / 1.08⁵ = 78.41
小計: 459.58円
```

**第2段階（安定期の終価）**
```
5年目配当: 100 × 1.15⁵ = 201.14円
6年目配当（D₆）: 201.14 × 1.03 = 207.18円
安定期現価 = 207.18 / (0.08 - 0.03) = 4,143.60円
6年目割引値 = 4,143.60 / 1.08⁵ = 2,826.99円
```

**理論株価**
```
P₀ = 459.58 + 2,826.99 = 3,286.57円
```

### 実用性

✓ 現実の企業のライフサイクルをより反映
✓ 成長企業の評価に適している
✓ GGMより柔軟

✗ パラメータが増加（計算複雑化）
✗ 段階分けの主観性

## 3. 配当持続可能性の指標

### 配当性向 (Payout Ratio)

```
配当性向 = 配当金 / 利益 × 100%
```

#### 解釈

| 範囲 | 評価 | 説明 |
|------|------|------|
| 0-30% | 非常に安全 | 利益の大部分を内部留保に回し、事業投資や配当増加に柔軟に対応可能 |
| 31-50% | 安全 | バランスの取れた配当政策。業界平均的 |
| 51-70% | 注意 | 配当性向が上昇傾向。景気悪化に弱い可能性 |
| 71-100% | 危険 | 利益の大部分を配当に回す。カット・減配リスク高 |
| 100%超 | 極度の危険 | 利益以上の配当。借入金や資本取り崩しで賄われている |

### EPS成長（経常利益安定性）

```
EPS変動係数 = 標準偏差(EPS) / 平均(EPS)
```

| 係数 | 安定性 | リスク |
|------|--------|--------|
| 0.0-0.1 | 非常に安定 | 低い |
| 0.1-0.2 | 安定 | 低い |
| 0.2-0.5 | 中程度 | 中程度 |
| 0.5-1.0 | 変動大 | 高い |
| 1.0超 | 極度の変動 | 非常に高い |

### 配当カバー率 (Dividend Coverage Ratio)

```
配当カバー率 = キャッシュフロー / 配当金
```

- カバー率 > 2.0: 安全（キャッシュで2倍以上カバー）
- カバー率 1.0-2.0: 標準的
- カバー率 < 1.0: 危険（キャッシュで賄い切れない）

## 5. 配当再投資（DRIP）の効果

### 複利成長の計算

配当を再投資した場合の最終価値：

```
FV = PV × (1 + r)ⁿ + PMT × [((1 + r)ⁿ - 1) / r]

FV   = 最終価値
PV   = 初期投資額
PMT  = 年間配当（定額と仮定）
r    = 配当再投資利回り + 株価成長率
n    = 年数
```

### 実例

#### 前提
- 初期投資: 100万円（100株 × 10,000円）
- 配当: 500円/株
- 配当利回り: 5%
- 株価成長率: 3%
- 総合成長率: 8%

#### 10年後

配当再投資なし：
```
株数: 100株
株価: 10,000 × 1.03¹⁰ = 13,439円
評価額: 100 × 13,439 = 1,343,900円
利益: 343,900円
```

配当再投資（DRIP）：
```
総合成長率 8% で複利
FV = 1,000,000 × 1.08¹⁰ = 2,158,925円
利益: 1,158,925円
```

**DRIP効果**: 追加利益 815,025円（+237%）

### 複利効果の強さ

時間経過による成長倍率（7%年利の場合）：

| 年数 | 成長倍率 |
|------|---------|
| 10年 | 1.97倍 |
| 20年 | 3.87倍 |
| 30年 | 7.61倍 |
| 50年 | 29.46倍 |

**長期投資ほどDRIPの効果が顕著になります。**

## 6. 実践的な評価フレームワーク

### スコア表（総合評価）

企業の配当投資魅力度の総合スコア（0-100）：

| スコア | 評価 | アクション |
|--------|------|-----------|
| 80-100 | 強気買い | ポジション構築 |
| 60-79 | 買い | 積立買い |
| 40-59 | 中立 | 様子見 |
| 20-39 | 売り | 減らす検討 |
| 0-19 | 強気売り | 売却検討 |

### 重要な警告サイン

以下の場合は配当カットの可能性が高い：

1. **配当性向が急上昇** （前年比 +10%以上）
2. **EPS が減少トレンド** （前年比 -20%以下）
3. **キャッシュフロー がマイナス**
4. **配当カバー率 が 1.0 未満**
5. **連続増配が終わり、減配に転じた**
6. **業界が苦境に直面** （規制、競争環境の悪化）
7. **経営者の交代** で方針変更

## 8. 注意事項

### モデルの限界

1. **過去データへの依存**: 将来が過去と同じとは限らない
2. **パラメータの不確定性**: g, r の推定は主観的
3. **環境変化への非対応**: 急激な産業構造変化には弱い
4. **配当政策の変更**: 経営方針の急転変が反映されない

### 使用上の注意

- **単一モデルに依存しない**: 複数の評価方法を組み合わせる
- **定期的な再評価**: 新情報が出たら即座に再計算
- **マージン・オブ・セーフティ**: 理論値より30-50%割安での購入を目指す
- **分散投資**: 配当企業のみに偏らない
