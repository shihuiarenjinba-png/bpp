# 配当分析スキル (Dividend Analysis Skill)

## 概要

このスキルは、株式の配当金データを詳細に分析し、投資家が配当戦略を評価するための包括的なレポートを生成します。

## 使い方

### ステップ1: データの準備

以下の列を含むCSVファイルを用意してください:

| 列名 | 説明 | 例 |
|------|------|-----|
| 銘柄 | 企業名または株式ティッカー | トヨタ、Toyota、7203 |
| 年度 | 配当が支払われた年度 | 2020, 2021, 2022 |
| 配当金額 | 1株当たりの配当金（円またはドル） | 100, 110, 120 |
| EPS | 1株当たり利益（Earnings Per Share） | 500, 550, 600 |
| 株価 | 配当支払い時点の株価 | 2500, 2750, 3000 |

**CSVサンプル:**
```
銘柄,年度,配当金額,EPS,株価
トヨタ,2018,200,800,2100
トヨタ,2019,210,820,2200
トヨタ,2020,200,750,2300
トヨタ,2021,220,900,2500
トヨタ,2022,250,950,2700
トヨタ,2023,270,1000,2900
```

### ステップ2: スクリプトの実行

```bash
python scripts/dividend_analyzer.py <入力CSVパス> <出力Excelパス>
```

**例:**
```bash
python scripts/dividend_analyzer.py ./data/dividend_data.csv ./reports/dividend_analysis_report.xlsx
```

### ステップ3: レポートの確認

生成されたExcelファイルには以下のシートが含まれます:

1. **サマリー**: 企業別の重要指標の概要
2. **配当履歴分析**: 年別の配当金、利回り、成長率の詳細分析
3. **DGM評価**: ゴードン成長モデルによる理論値評価
4. **再投資シミュレーション**: 配当再投資（DRIP）による10年・20年・30年の成長予測
5. **持続可能性**: 配当の持続可能性スコアと警告指標

## 計算される指標

### 配当利回り (Dividend Yield)
配当金額を現在の株価で割った比率です。投資からのキャッシュ利回りを示します。

```
配当利回り (%) = (年間配当金 / 株価) × 100
```

### 配当性向 (Payout Ratio)
利益に対する配当金の割合です。企業がどの程度の利益を配当に回しているかを示します。

```
配当性向 (%) = (年間配当金 / EPS) × 100
```

### 配当成長率 (Dividend Growth Rate - CAGR)
複数年にわたる配当金の年率成長率です。

```
CAGR (%) = [(最終年度配当金 / 開始年度配当金) ^ (1/年数) - 1] × 100
```

### 連続増配年数 (Years of Consecutive Increases)
配当金が前年比で増加した連続年数です。安定した配当成長を示す指標です。

### ゴードン成長モデル評価 (Gordon Growth Model / DDM)
配当割引モデルで理論上の企業価値を計算します。

```
理論株価 = D₁ / (r - g)

D₁: 次年度予想配当
r: 要求利回り（一般的に5-10%）
g: 永続成長率（配当成長率から推定）
```

### 配当再投資シミュレーション (DRIP Simulation)
配当を自動で再投資した場合の10年・20年・30年後の価値を計算します。

```
将来価値 = 初期投資 × (1 + r)^n + 定期配当の現価
```

### 配当の持続可能性スコア (Dividend Sustainability Score)
配当性向とEPS安定性から、配当の持続可能性を0-100のスコアで評価します。

- **スコア 80-100**: 非常に持続可能（安全性が高い）
- **スコア 60-79**: 比較的持続可能（通常レベル）
- **スコア 40-59**: 中程度のリスク（監視が必要）
- **スコア 20-39**: 高リスク（カット可能性あり）
- **スコア 0-19**: 非常に高リスク（カット警告）

## グラフの説明

### 配当トレンドライン
時系列で配当金額の推移を表示します。右肩上がりのトレンドは増配を示唆します。

### 利回り対株価チャート
配当利回りと株価の関係を散布図で表示します。利回りの魅力度を視覚的に評価できます。

### 再投資成長予測
DRIP（配当再投資）を実施した場合の複利成長を30年分表示します。時間の力の効果を示します。

## 注意事項

1. **過去データの制限**: この分析は過去のデータに基づいています。将来の配当が保証されるものではありません。
2. **カットの可能性**: 経済状況の悪化により、企業は配当をカットする可能性があります。
3. **税金**: 配当税は国や地域によって異なります。アフターテックス利回りを考慮してください。
4. **成長率の仮定**: DGMとDRIPシミュレーションで使用される成長率は歴史的データに基づいています。

## トラブルシューティング

### CSVファイルが読み込めない場合
- 文字エンコーディングが UTF-8 であることを確認してください
- 列名が正確に「銘柄」「年度」「配当金額」「EPS」「株価」であることを確認してください
- 数値列にカンマや不正な文字が含まれていないか確認してください

### 分析結果がおかしい場合
- 入力データに0や負の値が含まれていないか確認してください
- 年度が時系列順に並んでいることを確認してください
- データポイント数が最低3年以上あることを確認してください
