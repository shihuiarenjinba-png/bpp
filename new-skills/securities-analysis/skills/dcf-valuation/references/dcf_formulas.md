# DCFバリュエーション計算式リファレンス

## 概要
本ドキュメントは、割引キャッシュフロー(DCF)法に用いられる全ての計算式、その導出、および業界別のパラメータ範囲について説明します。

---

## 1. フリーキャッシュフロー(Free Cash Flow, FCF)

### 基本式
```
FCF = NOPAT + D&A - CapEx - ΔWC
```

### 各要素の説明

#### 1.1 NOPAT（税後営業利益）
```
営業利益 = 売上 - 営業費用
NOPAT = 営業利益 × (1 - 税率)
      = (売上 - 営業費用) × (1 - Tc)
```

- **営業利益**: EBIT（Earnings Before Interest and Taxes）とも呼ばれる
- **税率(Tc)**: 実効税率。通常、企業の申告所得税率と地方税を含める

#### 1.2 減価償却費・償却費(D&A: Depreciation & Amortization)
```
D&A = 有形資産の減価償却費 + 無形資産の償却費
```

- **性質**: 非現金費用（Depreciation）
- **役割**: 営業利益から控除されるが、キャッシュアウトフローではない
- **計算方法**: 
  - 定額法: D&A = 資産額 ÷ 耐用年数
  - 定率法: D&A = 残存簿価 × 償却率

#### 1.3 資本支出(Capital Expenditure, CapEx)
```
CapEx = 設備投資 + IT投資 + 買収等
```

- **特性**: キャッシュアウトフロー
- **予測方法**:
  - 過去のCapEx/売上比を使用
  - 業界平均の3-8%を使用
  - 経営計画に基づき個別予測

#### 1.4 運転資本変化(Change in Working Capital, ΔWC)
```
WC = 流動資産 - 流動負債
   = (売上債権 + 棚卸資産) - (買掛金 + 未払金)

ΔWC = WC(t) - WC(t-1)
```

- **売上債権**: 売上の○○日分（業界標準）
- **棚卸資産**: 売上原価の○○日分
- **買掛金**: 売上原価の○○日分
- **一般的な値**: ΔWC = 売上 × 2%-5%

### FCF計算の具体例

| 項目 | 金額 |
|------|------|
| 売上 | 1,000,000 |
| 営業費用 | 600,000 |
| 営業利益 | 400,000 |
| 税率 | 30% |
| NOPAT | 280,000 |
| (+) D&A | 50,000 |
| (-) CapEx | 80,000 |
| (-) ΔWC | 20,000 |
| **FCF** | **230,000** |

## 2. 加重平均資本コスト(WACC: Weighted Average Cost of Capital)

### 基本式
```
WACC = (E / (E + D)) × Re + (D / (E + D)) × Rd × (1 - Tc)
     = E_weight × Re + D_weight × Rd × (1 - Tc)
```

### 各要素の説明

#### 2.1 変数定義
- **E**: 株式の時価総額（Market Value of Equity）
- **D**: 負債の時価総額（Market Value of Debt）
- **Re**: 株式のコスト（Cost of Equity）
- **Rd**: 負債のコスト（Cost of Debt）
- **Tc**: 企業税率（Corporate Tax Rate）

#### 2.2 株式のコスト(Re)の計算

##### CAPM（資本資産価格モデル）
```
Re = Rf + β × (Rm - Rf)
   = リスクフリーレート + β × マーケットリスクプレミアム
```

- **Rf**: リスクフリーレート
  - 通常、長期国債利回りを使用
  - 日本: 10年物国債利回り（2-3%程度）
  - 米国: 10年物米国国債利回り（4-5%程度）

- **β**: ベータ（systematic risk）
  - 市場平均に対する当該企業の感応度
  - β = 1: 市場と同リスク
  - β > 1: 市場より高リスク
  - β < 1: 市場より低リスク
  - 業界別の代表企業のβを参照

- **Rm - Rf**: マーケットリスクプレミアム
  - 日本: 4-6%
  - 米国: 5-7%
  - 世界: 6-8%（長期平均）

##### Re計算の具体例
```
Re = 2.5% + 1.2 × 5.5%
   = 2.5% + 6.6%
   = 9.1%
```

#### 2.3 負債のコスト(Rd)

```
Rd = 利息支払額 / 平均負債残高

または

Rd = YTM(社債の利回り)
```

- **利息支払額**: 実績から計算
- **YTM**: 社債が上場している場合は市場利回りを使用

#### 2.4 ウェイト計算

##### 株式ウェイト
```
E_weight = E / (E + D)
```

##### 負債ウェイト
```
D_weight = D / (E + D)
```

#### 2.5 WACC計算の具体例

| 項目 | 値 |
|------|-----|
| 株式時価総額(E) | 50,000百万円 |
| 負債時価総額(D) | 20,000百万円 |
| 企業価値(E+D) | 70,000百万円 |
| 株式ウェイト | 71.4% |
| 負債ウェイト | 28.6% |
| 株式コスト(Re) | 10.0% |
| 負債コスト(Rd) | 5.0% |
| 税率(Tc) | 30% |
| **WACC** | **8.30%** |

```
WACC = 0.714 × 10.0% + 0.286 × 5.0% × (1 - 0.30)
     = 7.14% + 1.001%
     = 8.14%
```

## 3. ターミナルバリュー(Terminal Value)

### 3.1 ゴードン成長モデル(Gordon Growth Model)

```
TV = FCFn × (1 + g) / (WACC - g)
```

- **FCFn**: 最終予測年（n年目）のFCF
- **g**: 永続成長率（Terminal Growth Rate）
- **WACC**: 加重平均資本コスト

#### 永続成長率(g)の設定
- **経済成長率を参考**: 通常は名目GDP成長率と同程度
- **日本**: 2.0-2.5%
- **米国**: 2.5-3.0%
- **新興国**: 4.0-6.0%
- **過度な高さに注意**: g > WACC となるシナリオは避ける

#### ゴードン成長モデルの条件
```
必須: g < WACC（重要！）
```

- g >= WACC の場合、企業価値は無限大または負になる
- 現実的には g は WACC より3-5%低い値を設定

#### 計算例
```
FCF(5年目) = 230,000
WACC = 8.0%
g = 2.5%

TV = 230,000 × (1 + 0.025) / (0.08 - 0.025)
   = 230,000 × 1.025 / 0.055
   = 4,295,455
```

### 3.2 出口マルチプル法(Exit Multiple Method)

```
TV = EBITDA(n年目) × EV/EBITDA倍数
```

または

```
TV = 営業利益(n年目) × EV/営業利益倍数
```

#### 業界別のマルチプル目安

| 業界 | EV/EBITDA | EV/営業利益 |
|------|-----------|-----------|
| ソフトウェア | 8-12倍 | 15-20倍 |
| 小売業 | 6-10倍 | 10-15倍 |
| 製造業 | 5-8倍 | 8-12倍 |
| インフラ | 8-12倍 | 12-18倍 |
| 金融 | 3-6倍 | 8-12倍 |

#### 計算例
```
EBITDA(5年目) = 320,000
EV/EBITDA = 9倍

TV = 320,000 × 9 = 2,880,000
```

### 3.3 ゴードン成長モデル vs 出口マルチプル法

| 項目 | ゴードン成長 | 出口マルチプル |
|------|----------|----------|
| 適用範囲 | 成熟企業 | 成長段階・成熟企業 |
| パラメータ感応度 | 高（gとWACC） | 低（マルチプルの安定性） |
| 計算複雑度 | 低 | 低 |
| 市場データ依存 | 低 | 高（業界マルチプル） |

## 4. 企業価値(Enterprise Value)計算

### 基本式
```
EV = Σ(FCFt / (1 + WACC)^t) + TV / (1 + WACC)^n
   = PV(FCF) + PV(TV)
```

- **PV(FCF)**: 予測期間のFCFの現在価値合計
- **PV(TV)**: ターミナルバリューの現在価値

### 計算ステップ

#### ステップ1: 各年のFCFを現在価値に割引
```
PV(FCFt) = FCFt / (1 + WACC)^t
```

#### ステップ2: PV(FCF)の合計を計算
```
PV(FCF) = Σ PV(FCFt)  for t = 1 to n
```

#### ステップ3: ターミナルバリューを現在価値に割引
```
PV(TV) = TV / (1 + WACC)^n
```

#### ステップ4: 企業価値を計算
```
EV = PV(FCF) + PV(TV)
```

### 計算例

| 年度 | FCF | 割引率 | PV(FCF) |
|------|-----|--------|---------|
| 1 | 230,000 | 0.926 | 212,980 |
| 2 | 245,000 | 0.857 | 209,965 |
| 3 | 260,000 | 0.794 | 206,440 |
| 4 | 275,000 | 0.735 | 202,125 |
| 5 | 290,000 | 0.681 | 197,490 |
| **合計** | | | **1,029,000** |
| ターミナルバリュー | 4,295,455 | 0.681 | 2,925,155 |
| **企業価値(EV)** | | | **3,954,155** |

### EV の内訳（ターミナルバリューの割合）
```
TV / EV × 100% = 2,925,155 / 3,954,155 × 100% = 73.9%
```

通常、EV の60-80% がターミナルバリューで占められます。

## 5. 株価計算

### 基本式
```
株価 = (企業価値 - 負債 + 現金) / 発行済株式数
     = 株主価値 / 発行済株式数
```

### 各要素の説明

#### 5.1 企業価値(EV)からの調整
```
株主価値 = EV - 負債 + 現金
         = EV - Net Debt
```

- **Net Debt（ネット負債）**: Debt - Cash
  - 現金が負債を上回る場合は負の値
  - 例: 負債1,000 - 現金200 = ネット負債800

#### 5.2 発行済株式数(Shares Outstanding)
- 普通株式数
- オプション行使による希薄化を考慮（希薄化株式数を使用）
- Treasury stock（自社株）を除外

### 計算例
```
企業価値          3,954,155百万円
(-) 負債            200,000百万円
(+) 現金             150,000百万円
= 株主価値          3,904,155百万円

発行済株式数         100,000万株

株価 = 3,904,155 / 100,000 = 39,042円
```

## 6. 感度分析(Sensitivity Analysis)

### 6.1 目的
- WACC と永続成長率の変化に対する企業価値の感応度を測定
- 異なるシナリオでの企業価値レンジを把握
- リスク評価と意思決定の参考

### 6.2 WACC 感度（株式コスト(Re)の変化）
```
Re = Rf + β × (Rm - Rf)
```

Re の変化要因：
- **リスクフリーレートの上昇**: Rf が1% 上昇 → Re は1% 上昇
- **ベータの変化**: β が0.1 上昇 → Re は(Rm - Rf) × 0.1 上昇
  - (Rm - Rf) = 5% の場合、Re は0.5% 上昇

#### WACC 感度分析の範囲
```
通常の範囲: WACC ± 2% (中央値を中心)

例：中央値 8.0% の場合
    6.0% - 10.0% の範囲で感度分析
```

### 6.3 永続成長率(g)感度
```
g の変化要因：
- インフレ率の変化
- 産業成長率の変化
- 企業の競争優位の喪失・獲得
```

#### 成長率感度分析の範囲
```
保守的シナリオ: 0.0%-1.0%
基本シナリオ: 2.0%-2.5%
楽観的シナリオ: 3.0%-4.0%
```

### 6.4 感度分析マトリックス

| WACC / g | 0.0% | 1.0% | 2.0% | 2.5% | 3.0% |
|----------|------|------|------|------|------|
| **6.0%** | 3,833 | 4,127 | 4,501 | 4,782 | 5,167 |
| **7.0%** | 3,542 | 3,721 | 3,954 | 4,110 | 4,312 |
| **8.0%** | 3,292 | 3,416 | 3,579 | 3,682 | 3,822 |
| **9.0%** | 3,073 | 3,167 | 3,290 | 3,354 | 3,438 |
| **10.0%** | 2,878 | 2,954 | 3,050 | 3,104 | 3,170 |

#### 読み方
- 中央値（WACC 8.0%, g 2.5%）: 3,682（企業価値）
- 最大値（保守的WACC、高成長g）: 上左が最大
- 最小値（高WACC、低成長g）: 下右が最小

## 8. よくある間違いと留意点

### 8.1 WACC計算時の誤り

#### 間違い: 帳簿価額(Book Value)を使用
```
❌ 誤: WACC = (BVE / (BVE + BVD)) × Re + ...
✓ 正: WACC = (MVE / (MVE + MVD)) × Re + ...
```

**理由**: 時価総額は現在の市場評価を反映し、将来のキャッシュフロー割引に適合

#### 間違い: 負債コストに対して税効果を忘れる
```
❌ 誤: WACC = E_weight × Re + D_weight × Rd
✓ 正: WACC = E_weight × Re + D_weight × Rd × (1 - Tc)
```

**理由**: 利息は損金算入できるため、実効的なコストは低下

### 8.2 FCF計算時の誤り

#### 間違い: 営業キャッシュフロー(OCF)を使用
```
❌ 誤: FCF = OCF - CapEx
✓ 正: FCF = NOPAT + D&A - CapEx - ΔWC
```

**理由**: DCFは事業の経済的価値を反映する必要があり、財務取引を除外

#### 間違い: 過去のD&Aをそのまま使用
```
❌ 誤: FCF = NOPAT + 過去D&A - CapEx - ΔWC
✓ 正: FCF = NOPAT + 予測D&A - CapEx - ΔWC
```

**理由**: 将来の設備投資計画に合わせて予測D&Aを推定

### 8.3 永続成長率(g)の設定誤り

#### 間違い: 過度に高い成長率を設定
```
❌ 誤: g = 5.0%（先進国経済の成長率を上回る）
✓ 正: g = 2.0%-3.0%（名目GDP成長率と同程度）
```

**理由**: 
- 企業が長期で経済全体の成長率を上回り続けることは難しい
- 競争による利益縮小を考慮

#### 間違い: g >= WACC のシナリオ
```
❌ 誤: WACC = 7.0%, g = 7.5%
✓ 正: WACC = 8.0%, g = 2.5%（WACC > g）
```

**理由**: 数学的にも経済学的にも不合理（企業価値が無限大）

### 8.4 感度分析の活用誤り

#### 間違い: 感度分析結果を点推定として使用
```
❌ 誤: 「企業価値は3,500です」（感度分析の1つのセルの値）
✓ 正: 「企業価値は3,200-4,200の範囲で、基本シナリオは3,680です」
```

**理由**: 複数のシナリオの結果から妥当なレンジを判断する

## 10. チェックリスト

DCF分析実施時の確認項目：

- [ ] 入力データはすべて時価（Book Value ではない）か？
- [ ] WACC > 永続成長率 の条件を満たしているか？
- [ ] FCF の負の値が連続していないか？
- [ ] D&A は将来の投資計画と整合的か？
- [ ] CapEx は業界平均と比較して妥当か？
- [ ] 感度分析で複数シナリオを検討したか？
- [ ] ターミナルバリューの割合は60-80% か？
- [ ] 出口マルチプルは業界平均と整合的か？
- [ ] 税率は実効税率を使用しているか？
- [ ] 株式数の希薄化を考慮したか？
