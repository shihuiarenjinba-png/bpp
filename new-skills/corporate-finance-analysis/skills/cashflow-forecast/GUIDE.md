# キャッシュフロー予測スキル

企業の資金繰り管理と将来的な流動性リスクを評価するための包括的な予測ツールです。

## 概要

このスキルは、過去のキャッシュフロー実績から以下の分析を自動実行します：

- **実績分析**: 過去のキャッシュフロー傾向抽出
- **季節性検出**: 月次パターンと周期性の特定
- **マルチシナリオ予測**: 楽観/基本/悲観の3シナリオで12ヶ月間の予測
- **資金繰り表**: 営業CF・投資CF・財務CFの内訳と累積現金ポジション
- **リスク診断**: 連続した負CF月、6ヶ月以下の現金ランウェイなどの警告フラグ

## 使用方法

### ステップ1: データ準備

以下の形式のCSVファイルを用意してください：

```
日付,営業CF,投資CF,財務CF
2023-01,5000000,−2000000,1000000
2023-02,4500000,−1500000,0
...
```

**カラム要件:**
- `日付`: YYYY-MM形式
- `営業CF`: 営業活動によるキャッシュフロー（数値）
- `投資CF`: 投資活動によるキャッシュフロー（数値）
- `財務CF`: 財務活動によるキャッシュフロー（数値）

### ステップ2: スクリプト実行

```bash
python scripts/cashflow_forecaster.py \
  --input データ.csv \
  --output 予測結果.xlsx \
  --initial-cash 初期現金ポジション
```

**パラメータ:**
- `--input`: 入力CSVファイルパス（必須）
- `--output`: 出力Excelファイルパス（必須）
- `--initial-cash`: 開始時の現金ポジション（デフォルト: 10,000,000）

### ステップ3: 結果確認

出力Excelファイルには以下のシートが含まれます：

| シート名 | 内容 |
|---------|------|
| **サマリー** | 予測結果の要約、主要指標、リスク評価 |
| **実績分析** | 過去12ヶ月のCF実績と傾向分析 |
| **季節性パターン** | 月次パターン指数と周期性分析 |
| **シナリオ予測** | 楽観/基本/悲観シナリオの12ヶ月予測 |
| **資金繰り表** | 詳細な資金繰り計算表（営業/投資/財務別） |
| **警告フラグ** | リスク診断結果と改善アクション |

## 出力ファイル

### Excelレポート

- **複数チャート**: 実績vs予測、季節パターン、シナリオ比較、資金ショートリスク
- **詳細データシート**: 月次実績、予測値、キャッシュポジション
- **日本語ダッシュボード**: 経営層向けの分かりやすい視覚化

### 数値出力

- 現金ランウェイ（月数）
- バーンレート（月平均）
- リスク度（スコア0-100）
- 推奨現金保有水準

## 主要指標の解釈

### キャッシュランウェイ

現在の現金残高を月平均バーンレートで割った値。資金繰り悪化時に何ヶ月持つかを示します。

- **6ヶ月以上**: 健全
- **3～6ヶ月**: 注意
- **3ヶ月未満**: 危機的

### バーンレート

月平均のキャッシュ消費額。営業CFが負の企業で重要な指標です。

```
バーンレート = 直近3ヶ月の平均負CF額
```

### 季節性指数

月次の標準化パターン。1.0を基準として、高い月・低い月を特定します。

## 方法論

詳細は `references/forecasting_methods.md` を参照してください。

- **移動平均法**: 短期トレンド抽出
- **トレンド外挿**: 長期傾向の予測
- **季節分解**: Monthly × Trendでシナリオを構築
- **シナリオ分析**: ±10%～±20%の範囲で複数シナリオを作成
