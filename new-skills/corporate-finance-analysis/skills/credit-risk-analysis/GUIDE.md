# 信用リスク分析スキル

## 概要

このスキルは、企業の信用リスク（倒産リスク、デフォルトリスク）を定量的に評価する完全なツールセットです。Altman Z-Score と Piotroski F-Score の 2 つの主要な信用スコアリングモデルを実装し、流動性分析、利息カバレッジ分析、総合信用格付けを提供します。

## 使用方法

### 1. 入力データの準備

CSV または Excel（.xlsx）形式で、以下の財務情報を含むファイルを用意してください：

**必須列（日本語名または英語名）：**
- `企業名` / `Company`: 企業名
- `期間` / `Period`: 会計期間（例：2024-03-31）
- `流動資産` / `Current Assets`: 流動資産
- `固定資産` / `Fixed Assets`: 固定資産
- `総資産` / `Total Assets`: 総資産
- `流動負債` / `Current Liabilities`: 流動負債
- `長期負債` / `Long-term Liabilities`: 長期負債
- `総負債` / `Total Liabilities`: 総負債
- `売上高` / `Sales`: 売上高
- `営業利益` / `EBIT`: 営業利益
- `純利益` / `Net Income`: 純利益
- `営業キャッシュフロー` / `Operating Cash Flow`: 営業キャッシュフロー
- `市場時価総額` / `Market Value of Equity`: 市場時価総額（株式の時価総額）
- `売上原価` / `Cost of Goods Sold`: 売上原価
- `前期総資産` / `Prior Year Total Assets`: 前年度総資産（トレンド分析用）
- `前期営業利益率` / `Prior Year ROA`: 前年度営業利益率（トレンド分析用）

### 2. スクリプト実行

```bash
python scripts/credit_risk_analyzer.py \
    --input /path/to/financial_data.xlsx \
    --output /path/to/credit_risk_report.xlsx
```

### 3. 出力レポート

Python スクリプトは以下の内容を含む Excel レポートを生成します：

| シート名 | 内容 |
|---------|------|
| **サマリー** | 総合信用格付け、主要スコア、リスク評価 |
| **Altman Z-Score** | Z-Score計算値、安全性評価（3つのゾーン） |
| **Piotroski F-Score** | 9つの財務指標の個別評価 |
| **流動性分析** | 流動比率、速動比率、現金比率、総合流動性スコア |
| **総合信用格付け** | AAA～D の信用格付けスケール |
| **トレンド分析** | 期間ごとのスコア推移、リスク傾向 |

## モデル説明

### Altman Z-Score

企業の倒産リスクを評価する多変量モデルです。以下の5つの財務指標の加重合計により計算されます。

**計算式（製造業向け）：**
```
Z = 1.2×X₁ + 1.4×X₂ + 3.3×X₃ + 0.6×X₄ + 1.0×X₅

X₁ = 運転資本 / 総資産
X₂ = 利益剰余金 / 総資産
X₃ = 営業利益 / 総資産
X₄ = 時価総額 / 総負債
X₅ = 売上高 / 総資産
```

**安全性ゾーン：**
- **Safe Zone（安全）**: Z > 2.99 - 倒産リスク低い
- **Grey Zone（警戒）**: 1.81 ≤ Z ≤ 2.99 - 要注視
- **Distress Zone（危機）**: Z < 1.81 - 倒産リスク高い

**非製造業向け Z''-Score：** 非製造業の場合、X₄の係数と基準値が異なります。

### Piotroski F-Score

企業の財務力を 9 つの指標で評価（各1点、合計9点）するシンプルで高精度なモデルです。

**9つの評価項目：**
1. **ROA（自己資本利益率）** ≥ 0% → 1点
2. **営業キャッシュフロー** ≥ 0 → 1点
3. **ΔROA（ROA の改善）** > 0 → 1点
4. **キャッシュフロー質** （営業CF > 純利益）→ 1点
5. **Δ負債比率（低下）** < 0 → 1点
6. **Δ流動比率（改善）** > 0 → 1点
7. **新株発行なし** → 1点
8. **Δ売上総利益率（改善）** > 0 → 1点
9. **Δ資産回転率（改善）** > 0 → 1点

**総合評価：**
- **8～9点**: 優秀（AAA～AA相当）
- **6～7点**: 良好（A相当）
- **4～5点**: 平均（BBB相当）
- **2～3点**: 要注視（BB以下相当）
- **0～1点**: 危険（B～D相当）

### 流動性分析

企業の短期支払能力を多次元で評価します。

**指標：**
- **流動比率** = 流動資産 / 流動負債（目安: > 1.5）
- **速動比率** = （流動資産 - 在庫）/ 流動負債（目安: > 1.0）
- **現金比率** = 現金・現金同等物 / 流動負債（目安: > 0.2）
- **総合流動性スコア** = 上記3つの加重平均

### 信用格付けスケール

Z-Score、F-Score、流動性スコアを統合して、AAA～D の信用格付けを算出します。

| 格付け | 意味 | リスク |
|--------|------|--------|
| **AAA** | 最上級 | 極めて低い |
| **AA** | 上級 | 非常に低い |
| **A** | 上位中級 | 低い |
| **BBB** | 中級 | 中程度 |
| **BB** | 下位中級 | やや高い |
| **B** | 低格付け | 高い |
| **CCC** | 投機的 | 非常に高い |
| **CC** | 非常に投機的 | 非常に高い |
| **C** | 極度に投機的 | 極めて高い |
| **D** | デフォルト状態 | 実現 |

## データ例

```csv
企業名,期間,流動資産,固定資産,総資産,流動負債,長期負債,総負債,売上高,営業利益,純利益,営業キャッシュフロー,市場時価総額,売上原価,前期総資産,前期営業利益率
太郎電子,2024-03-31,5000,8000,13000,3000,4000,7000,10000,1500,900,1200,15000,6000,12500,0.12
```

## 注意事項

- **入力データの正確性**: 財務データは公式な財務諸表から抽出してください
- **通貨単位**: 全ての数値は同一通貨単位（百万円など）で統一してください
- **市場時価総額**: 上場企業の場合は時価総額を使用してください。非上場企業の場合は推定値または簿価を使用できます
- **周期性**: 財務データは同一の会計周期で準備してください
- **モデルの限界**: Altman Z-Score と Piotroski F-Score は参考指標です。業界特性や景気動向など定性的要因も考慮してください
