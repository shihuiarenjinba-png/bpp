# 信用リスク分析モデル リファレンス

## 1. Altman Z-Score モデル

### 概要
Edward Altman が1968年に開発した企業の倒産予測モデル。財務比率の線形結合により
倒産リスクを定量的に評価する。

### 計算式

#### 製造業向け（Original Z-Score）
```
Z = 1.2×X1 + 1.4×X2 + 3.3×X3 + 0.6×X4 + 1.0×X5
```

| 変数 | 定義 | 意味 |
|------|------|------|
| X1 | 運転資本 / 総資産 | 短期流動性 |
| X2 | 利益剰余金 / 総資産 | 累積収益力 |
| X3 | EBIT / 総資産 | 営業収益力 |
| X4 | 株式時価総額 / 総負債 | 資本構成 |
| X5 | 売上高 / 総資産 | 資産効率 |

#### 判定ゾーン
| ゾーン | Z-Score | 解釈 |
|--------|---------|------|
| 安全圏 (Safe) | Z > 2.99 | 倒産リスクが低い |
| グレーゾーン | 1.81 ≤ Z ≤ 2.99 | 要注意・監視が必要 |
| 危険圏 (Distress) | Z < 1.81 | 倒産リスクが高い |

#### 非製造業向け（Z''-Score）
```
Z'' = 6.56×X1 + 3.26×X2 + 6.72×X3 + 1.05×X4
```
- X4 = 株主資本簿価 / 総負債（時価総額ではなく簿価を使用）
- X5（売上高/総資産）は業種バイアスを避けるため除外

#### 非製造業の判定ゾーン
| ゾーン | Z''-Score | 解釈 |
|--------|-----------|------|
| 安全圏 | Z'' > 2.60 | 倒産リスクが低い |
| グレーゾーン | 1.10 ≤ Z'' ≤ 2.60 | 要注意 |
| 危険圏 | Z'' < 1.10 | 倒産リスクが高い |

### 限界と注意点
- 米国の製造業データで構築されたモデルであり、日本企業や非製造業への適用には注意
- 会計基準の違い（IFRS vs US-GAAP vs 日本基準）が結果に影響
- 新興企業や高成長企業では利益剰余金が小さく、過度に低く評価される傾向
- 定期的なモニタリング（四半期ごと）が推奨

## 2. Piotroski F-Score モデル

### 概要
Joseph Piotroski が2000年に発表した財務健全性スコアリングモデル。
9つのバイナリ指標（0または1）の合計で0〜9点のスコアを算出する。

### 9つの判定基準

#### 収益性（Profitability）: 4項目
| # | 基準 | 条件（1点） |
|---|------|-------------|
| F1 | ROA | 当期ROA > 0 |
| F2 | 営業キャッシュフロー | 当期営業CF > 0 |
| F3 | ROA改善 | 当期ROA > 前期ROA |
| F4 | CF品質 | 営業CF > 当期純利益（発生主義利益より現金利益が大きい） |

#### 財務レバレッジ・流動性（Leverage/Liquidity）: 3項目
| # | 基準 | 条件（1点） |
|---|------|-------------|
| F5 | レバレッジ改善 | 当期長期負債比率 < 前期長期負債比率 |
| F6 | 流動比率改善 | 当期流動比率 > 前期流動比率 |
| F7 | 希薄化なし | 新株発行なし（発行済株式数が増加していない） |

#### 効率性（Operating Efficiency）: 2項目
| # | 基準 | 条件（1点） |
|---|------|-------------|
| F8 | 粗利率改善 | 当期粗利率 > 前期粗利率 |
| F9 | 資産回転率改善 | 当期総資産回転率 > 前期総資産回転率 |

### スコアの解釈
| スコア | 評価 | 解釈 |
|--------|------|------|
| 8-9 | 優良 (Strong) | 財務健全性が高く、投資候補として有望 |
| 5-7 | 普通 (Moderate) | 一定の健全性があるが改善の余地あり |
| 0-4 | 脆弱 (Weak) | 財務面に懸念があり、注意が必要 |

### 活用方法
- バリュー投資のスクリーニングに特に有効（低PBR銘柄から高F-Scoreを選別）
- 単独使用よりも他の指標と組み合わせることが推奨
- 前年比較が基本のため、最低2期分の財務データが必要

## 3. 流動性分析

### 主要指標

#### 流動比率（Current Ratio）
```
流動比率 = 流動資産 / 流動負債
```
- 基準: 200%以上が理想、100%未満は要警戒
- 業界により大きく異なる（小売業は低め、製造業は高め）

#### 当座比率（Quick Ratio）
```
当座比率 = (流動資産 - 棚卸資産 - 前払費用) / 流動負債
```
- 基準: 100%以上が理想
- 流動比率より厳格な短期支払能力の指標

#### 現金比率（Cash Ratio）
```
現金比率 = (現金及び預金 + 有価証券) / 流動負債
```
- 最も保守的な流動性指標
- 基準: 20%以上が望ましい

### 複合流動性スコア
```
流動性スコア = 0.4 × (流動比率スコア) + 0.35 × (当座比率スコア) + 0.25 × (現金比率スコア)
```
各スコアは0〜100の範囲に正規化（業界基準との比較）

## 4. インタレスト・カバレッジ・レシオ

### 計算式
```
ICR = EBIT / 支払利息
```

### 判定基準
| ICR | 評価 | 解釈 |
|-----|------|------|
| > 5.0 | 極めて安全 | 利払い能力に十分な余裕 |
| 3.0 - 5.0 | 安全 | 良好な利払い能力 |
| 1.5 - 3.0 | 注意 | 業績悪化時にリスク |
| 1.0 - 1.5 | 警戒 | 利払いに余裕なし |
| < 1.0 | 危険 | 利払い困難な状態 |

---

## 5. 総合信用格付けマッピング

### スコア算出方法
各モデルのスコアを統合して総合信用格付けを算出する。

```
総合スコア = 0.35 × Z-Score正規化 + 0.25 × F-Score正規化 + 0.25 × 流動性スコア + 0.15 × ICRスコア
```

### 格付け対応表
| 格付け | 総合スコア | 解釈 |
|--------|-----------|------|
| AAA | 90-100 | 最高の信用力 |
| AA | 80-89 | 極めて高い信用力 |
| A | 70-79 | 高い信用力 |
| BBB | 60-69 | 十分な信用力（投資適格下限） |
| BB | 50-59 | 投機的要素あり |
| B | 40-49 | 投機的 |
| CCC | 30-39 | 信用リスクが高い |
| CC | 20-29 | 極めて高い信用リスク |
| C | 10-19 | デフォルト間近 |
| D | 0-9 | デフォルト状態 |

## 6. Merton モデル（概念解説）

### 概要
Robert Merton が1974年に提唱した構造的信用リスクモデル。
企業の株式を「企業資産に対するコールオプション」と見なす。

### 基本的な考え方
- 企業の資産価値が負債の額面を下回った場合にデフォルトが発生
- Black-Scholesのオプション価格理論を応用
- デフォルト確率（PD）と信用スプレッドを理論的に算出

### 計算式（概念）
```
E = V × N(d1) - D × e^(-rT) × N(d2)

d1 = [ln(V/D) + (r + σ²/2)×T] / (σ√T)
d2 = d1 - σ√T

デフォルト確率 = N(-d2)
```
- E: 株式時価総額、V: 資産価値、D: 負債額面
- r: リスクフリーレート、σ: 資産ボラティリティ、T: 負債の満期

### 注意点
- 実務での適用には資産ボラティリティの推定が必要（直接観測不可）
- 本スキルでは簡易版（KMV風のDistance to Default）を参考値として算出
- 詳細な実装には専門的な金融工学の知識が必要

## 7. 各モデルの比較と使い分け

| 項目 | Z-Score | F-Score | 流動性分析 | Merton |
|------|---------|---------|-----------|--------|
| 目的 | 倒産予測 | 財務健全性 | 短期支払能力 | デフォルト確率 |
| 必要データ | 1期分 | 2期分 | 1期分 | 市場データ |
| 対象 | 上場企業 | 全企業 | 全企業 | 上場企業 |
| 精度 | 高（2年以内） | 中 | 中 | 高（理論的） |
| 複雑さ | 低 | 低 | 低 | 高 |
| 推奨用途 | スクリーニング | バリュー投資 | 日常モニタリング | 高度な分析 |

### 推奨される組み合わせ
1. **基本分析**: Z-Score + F-Score + 流動性分析
2. **投資判断**: Z-Score + F-Score（バリュー投資スクリーニング）
3. **与信管理**: Z-Score + 流動性分析 + ICR
4. **高度分析**: 上記全て + Merton モデル参考値
