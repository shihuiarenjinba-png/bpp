# 比較財務分析スキル

## 概要

複数企業の財務データを包括的に比較分析します。コモンサイズ分析、成長率比較、主要指標ランキング、強み弱み分析を実施し、Excel形式の詳細レポートを生成します。

## 使用方法

### 入力データ形式

#### パターン1: 単一CSVまたはExcelファイル（推奨）
- 複数企業のデータを1つのファイルに統合
- 最初の列に企業名（会社名）を記載
- 以降の列に各指標を配置

**必須列:**
- 会社名 (Company Name)
- 売上 (Revenue)
- 営業利益 (Operating Income)
- 純利益 (Net Income)
- 総資産 (Total Assets)
- 株主資本 (Shareholders' Equity)
- 流動資産 (Current Assets)
- 流動負債 (Current Liabilities)

**推奨データ:**
- 複数会計年度（比較期間）
- 売上原価、営業費用などの詳細損益データ
- 各資産・負債項目

#### パターン2: 複数ファイル
- 企業ごとに別ファイルを用意
- ファイル名に企業名を含める
- 各ファイルのシート名は統一

### 実行コマンド

```bash
python scripts/comparative_analyzer.py \
  --input "/path/to/input_file.xlsx" \
  --output "/path/to/output_report.xlsx" \
  --company-column "会社名"
```

複数ファイルの場合:
```bash
python scripts/comparative_analyzer.py \
  --input "/path/to/folder/*.xlsx" \
  --output "/path/to/output_report.xlsx"
```

### 出力レポート構成

生成されるExcelファイルには以下のシートが含まれます:

1. **サマリー**: 主要指標の概要と最高・最低企業の表示
2. **コモンサイズPL**: 売上に対する百分率損益計算書
3. **コモンサイズBS**: 総資産に対する百分率貸借対照表
4. **成長率比較**: 年間成長率（YoY）とCAGR（複合年間成長率）
5. **指標ランキング**: ROE、ROA、利益率などの主要指標による順位付け
6. **総合評価**: 企業ごとの強み・弱み分析と総合スコア

### 分析対象指標

**収益性指標:**
- ROE（自己資本利益率）
- ROA（総資産利益率）
- 営業利益率
- 純利益率
- EBITDA利益率

**成長指標:**
- 売上成長率（YoY）
- 営業利益成長率（YoY）
- 純利益成長率（YoY）
- 売上CAGR（複合年間成長率）
- 純利益CAGR

**効率性指標:**
- 総資産回転率
- 流動比率
- 当座比率

**レバレッジ指標:**
- 負債比率
- 負債資本比率
- 利息カバー率

### 出力チャート

1. **主要指標比較グラフ**: 企業ごとの重要指標を並べたグループ棒グラフ
2. **レーダーチャート**: 複数企業の多次元指標を同時に表示
3. **成長軌跡グラフ**: 時系列での成長トレンド比較

## 分析方法

### コモンサイズ分析（百分率分析）

損益計算書の各項目を売上の百分率として表示し、企業規模の違いを調整して比較します。
- 売上原価率が異なれば営業効率が異なる
- 営業費用率から構造的なコスト差を識別
- 異なる規模の企業でも相対的な効率性を比較可能

貸借対照表では各項目を総資産の百分率として表示し、資本構成を比較します。

### ピアグループ選定

同業他社との比較には以下の基準を使用します:
- 同一産業・セクター
- 類似の事業規模（売上高規模）
- 同一国・地域（通貨・規制環境）
- 類似の事業モデル

### 成長率計算

**年間成長率（YoY）:**
```
YoY = (当年度 - 前年度) / 前年度 × 100%
```

**複合年間成長率（CAGR）:**
```
CAGR = (終了年度値 / 開始年度値) ^ (1/年数) - 1
```

3年以上のデータがあれば、CAGRは短期的な変動に左右されない成長トレンドを把握できます。

### 評価ギャップの解釈

**相対評価ギャップ:**
- ピアグループ平均との差分を確認
- 平均より高い項目 = 競争優位性がある領域
- 平均より低い項目 = 改善が必要な領域

**業界ベンチマーク:**
- 業界標準値との比較により、差別化要因を特定
- 規制環境や事業特性に応じた調整が必要

## 使用例

### 例1: 小売業3社の比較

入力: 過去3年間の財務データ（3社）
→ コモンサイズPLで原価率・営業効率を比較
→ 成長率比較で市場シェア拡大の動き分析
→ ROE・ROA比較で収益性の差を確認
→ レーダーチャートで全体像を可視化

### 例2: 製造業5社のベンチマーク分析

入力: 最新会計年度の財務データ（5社）
→ コモンサイズBS で資本構成の差を分析
→ 指標ランキングで各企業の相対位置を明確化
→ 総合評価で各社の強み・弱みマトリックスを作成

## 出力ファイルサイズと処理時間

- 小規模（企業数≤10、期間≤5年）: 数秒、ファイルサイズ 1-5MB
- 中規模（企業数11-30、期間≤10年）: 10-30秒、ファイルサイズ 5-15MB
- 大規模（企業数>30または複雑な計算）: 1-3分、ファイルサイズ 15-50MB

## 注意事項

1. **データ品質**: 売上高、利益、資産が負の値でないこと、単位が統一されていることを確認してください
2. **通貨統一**: 複数通貨の場合は事前に統一してください
3. **会計基準**: IFRS/US-GAAUPなど異なる基準の企業を比較する際は注記してください
4. **連結・単体**: 比較企業で同じベースを使用してください（全て連結決算など）
5. **異常値**: 1回限りの特別損益や重大な減損があれば除外を検討してください
