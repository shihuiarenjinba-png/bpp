## コモンサイズ分析

### 概要

コモンサイズ分析（百分率分析）は、異なる規模の企業を比較するための基本的な手法です。財務諸表の各項目を一定の基準額に対する百分率として表現し、企業規模の違いを調整します。

### 損益計算書のコモンサイズ分析

**基準：売上高 = 100%**

損益計算書のすべての項目を売上高で割り、百分率で表現します。

#### 計算式

```
コモンサイズ値（%）= （各項目 ÷ 売上高） × 100
```

#### 例

| 項目 | A企業（百万円） | A企業（%） | B企業（百万円） | B企業（%） |
|------|-----------------|-----------|-----------------|-----------|
| 売上 | 10,000 | 100.0 | 5,000 | 100.0 |
| 売上原価 | 6,000 | 60.0 | 3,000 | 60.0 |
| 営業費用 | 2,500 | 25.0 | 1,500 | 30.0 |
| 営業利益 | 1,500 | 15.0 | 500 | 10.0 |
| 純利益 | 1,000 | 10.0 | 250 | 5.0 |

#### 解釈ポイント

1. **原価率（売上原価 ÷ 売上）**
   - 製造業では50-70%が一般的
   - 低い → 製造効率が良い
   - 高い → 材料費またはスケール経済の問題

2. **営業費用率（営業費用 ÷ 売上）**
   - 販売・管理費の効率を示す
   - 企業規模が小さいほど高い傾向
   - 規模の経済（スケールメリット）が反映される

3. **営業利益率（営業利益 ÷ 売上）**
   - コア事業の収益性を示す
   - 業種ごとに標準値が異なる
   - 同業他社との比較に有効

4. **純利益率（純利益 ÷ 売上）**
   - 全体的な収益性を示す
   - 営業外損益や税金の影響を反映
   - 長期的な競争力を示す指標

### 貸借対照表のコモンサイズ分析

**基準：総資産 = 100%**

貸借対照表のすべての項目を総資産で割り、百分率で表現します。

#### 計算式

```
コモンサイズ値（%）= （各項目 ÷ 総資産） × 100
```

#### 例

| 資産項目 | A企業（百万円） | A企業（%） | B企業（百万円） | B企業（%） |
|---------|-----------------|-----------|-----------------|-----------|
| 流動資産 | 3,000 | 30.0 | 1,200 | 30.0 |
| 固定資産 | 7,000 | 70.0 | 2,800 | 70.0 |
| **総資産** | **10,000** | **100.0** | **4,000** | **100.0** |
| 流動負債 | 2,000 | 20.0 | 800 | 20.0 |
| 固定負債 | 3,000 | 30.0 | 1,200 | 30.0 |
| 株主資本 | 5,000 | 50.0 | 2,000 | 50.0 |

#### 解釈ポイント

1. **流動資産比率**
   - 短期的な支払い能力を反映
   - 小売業（高い）vs. 製造業（低い）
   - 過度に高い → 非効率な資金運用の可能性

2. **固定資産比率**
   - 事業性資産への投資を示す
   - 資本集約的事業（高い）vs. サービス業（低い）
   - 長期的な競争力の源泉

3. **負債比率（負債 ÷ 総資産）**
   - レバレッジ（財務杠杆）を示す
   - 業種によって適正水準が異なる
   - 金融危機への耐性を示す

4. **自己資本比率（株主資本 ÷ 総資産）**
   - 企業の財務安定性
   - 高い → 保守的で安定、低い → 積極的だが脆弱性がある
   - 国別・業種別に標準値がある

## ピアグループ選定

### ピアグループの定義

ピアグループとは、比較対象企業と業務内容・経営特性が類似した企業グループです。正確な比較分析には、適切なピアグループの選定が不可欠です。

### 選定基準

#### 1. 業種・セクター

```
同じ業種区分に属する企業を選定します。
• 大分類：製造業、小売業、サービス業など
• 中分類：自動車製造、食品製造など
• 小分類：乗用車製造、トラック製造など

詳細度が高いほど比較精度が向上します。
```

**例外:**
- グローバル企業は地域別の規制環境の違いを考慮
- 異業種への事業拡大している企業は複数ピアグループを使用

#### 2. 企業規模（売上高）

```
売上高が2倍～0.5倍程度の企業を選定します。

例：比較企業の売上が1,000億円の場合
推奨範囲：500億円～2,000億円
```

**理由:**
- 企業規模により経営指標が大きく変わる
- スケールメリット・デメリットが反映される
- 規制要件や上場企業基準も異なる

#### 3. 地域・国

```
基本的には同一国・地域の企業を選定します。
異なる場合は以下を調整：
• 通貨（原則、統一通貨に換算）
• 規制環境（金融規制、労働法など）
• 会計基準（IFRS vs. 現地基準）
• 税制（法人税率、優遇制度）
```

#### 4. 事業モデルの類似性

```
以下の要素が類似している企業を選定：

① 垂直統合度
   - 完全垂直統合型（原材料～販売まで）
   - 部分統合型（一部を外注）
   - 完全分離型（外注が大部分）

② 販売チャネル
   - B2B（企業向け）
   - B2C（消費者向け）
   - 直販 vs. 流通販売

③ 製品特性
   - 商品の生存期間
   - 技術集約度
   - カスタマイズの程度

④ 国際化度
   - 国内中心型
   - 地域展開型
   - グローバル展開型
```

### ピアグループ選定プロセス

```
1. 候補企業リストの作成
   ↓
2. 上記4基準で絞り込み
   ↓
3. 最小3社、推奨5-10社を選定
   ↓
4. 定性的な事業特性を確認
   ↓
5. ピアグループの妥当性を文書化
```

### ピアグループの修正

以下の場合はピアグループを再検討します：

- **異常値が多い**: 1社が極端に異なる値の場合、その企業を除外も検討
- **事業構造の変化**: M&A、事業売却などで比較可能性が低下
- **時系列変化**: 過去3年と現在でピアグループが変わる可能性

## 成長率計算

### 年間成長率（YoY: Year-over-Year）

#### 定義

前年度比での成長率。最新の1年間の成長率を示します。

#### 計算式

```
YoY = （当年度値 - 前年度値） / 前年度値 × 100%
```

#### 例

```
2024年度売上：1,200百万円
2023年度売上：1,000百万円

YoY = (1,200 - 1,000) / 1,000 × 100% = 20%
```

#### 特徴

✓ 最新の成長動向を反映
✓ 計算が簡単で解釈しやすい
✗ 1年間のデータのみ（変動しやすい）
✗ 異常値の影響を受けやすい

#### 用途

- 最新の経営状況の評価
- 市場環境の急激な変化の検出
- 短期的な経営判断

### 複合年間成長率（CAGR: Compound Annual Growth Rate）

#### 定義

複数年にわたる成長率を年率で平均化した指標。短期的な変動に左右されない、中長期的な成長トレンドを示します。

#### 計算式

```
CAGR = (終了時点の値 / 開始時点の値) ^ (1/年数) - 1
```

#### 例

```
2020年度売上：1,000百万円
2024年度売上：1,610百万円
年数：4年

CAGR = (1,610 / 1,000) ^ (1/4) - 1
     = 1.61 ^ 0.25 - 1
     = 1.125 - 1
     = 12.5%
```

#### 特徴

✓ 長期的なトレンドを反映
✓ 短期的な変動に強い
✓ 複数企業の成長を公平に比較
✗ 計算が複雑
✗ 最新データの急激な変化を過小評価

#### 用途

- 企業の持続的な成長力の評価
- 複数期間にわたる成長トレンドの分析
- ピアグループとの成長比較
- 投資判断の材料

### YoYとCAGRの使い分け

| 項目 | YoY | CAGR |
|------|-----|------|
| **時間軸** | 1年間 | 複数年（3年以上推奨） |
| **変動性** | 影響を受けやすい | 影響を受けにくい |
| **用途** | 短期評価 | 中長期評価 |
| **データ要件** | 2時点 | 3時点以上 |

#### 組み合わせ活用例

```
5年間の売上推移：1,000 → 1,100 → 900 → 1,300 → 1,500

YoY（2024年）= (1,500 - 1,300) / 1,300 = 15.4%
  → 直近1年は好調

CAGR（過去4年）= (1,500 / 1,000) ^ (1/4) - 1 = 10.7%
  → 長期的には安定成長

解釈：一時的な落ち込みはあったが、長期的には安定成長。
     最近さらに加速している可能性がある。
```

### 注意事項

#### 1. 異常値の処理

```
一度限りの特別な出来事（大型M&A、事業売却）の場合：
• YoYは実績のため調整しない
• CAGR計算では該当年度を除外も検討
```

#### 2. 通貨変動の影響

```
国際企業の場合：
• 現地通貨での成長率を確認
• 為替の影響を別途分析
• 必要に応じて為替調整後の成長率も計算
```

#### 3. 有機成長（オーガニック成長）

```
M&Aを含まない成長を「有機成長」と呼びます：
• 売上高成長 = 有機成長 + M&Aによる成長
• 経営力の評価には有機成長が重要
```

## 相対評価ギャップの解釈

### ギャップの概念

相対評価ギャップとは、分析対象企業の指標値と、ピアグループ平均値との差のことです。

### 計算方法

#### 絶対差分

```
ギャップ（絶対値） = 企業の指標値 - ピアグループ平均値
```

#### 相対差分（百分率ギャップ）

```
ギャップ（%） = （企業の指標値 - ピアグループ平均値）/ ピアグループ平均値 × 100%
```

#### 例

```
企業A の ROE = 15%
ピアグループ平均 ROE = 12%

絶対差分 = 15% - 12% = +3%ポイント
相対差分 = (15% - 12%) / 12% × 100% = +25%
```

### ギャップの解釈フレームワーク

#### 1. 正のギャップ（企業 > ピアグループ平均）

**意味：** 該当領域で競争優位がある

**例：ROE が高い場合**
- 自己資本をより効率的に活用している
- 利益創出力が強い
- 競合より投資に対するリターンが大きい

**推測される要因：**
- 経営効率の優位性
- 独自の事業モデル
- ブランド力による価格競争力
- 技術的優位性
- スケール経済の活用

#### 2. 負のギャップ（企業 < ピアグループ平均）

**意味：** 該当領域に改善の余地がある

**例：営業利益率が低い場合**
- コスト競争力が劣っている可能性
- 事業効率が劣っている
- 競合力強化が必要

**推測される要因：**
- 経営効率の劣後性
- スケールの小ささ
- 市場での立場の弱さ
- 技術的遅れ
- コスト構造の非効率性

### ギャップ分析のプロセス

#### ステップ1: ギャップの特定

```
各指標について、ピアグループ平均との差を計算・分析
↓
大きなギャップ（±20%以上）から優先的に検討
```

#### ステップ2: ギャップの要因分析

```
正のギャップの場合：
  ① その要因は持続可能か？
  ② 他企業が追い付く可能性は？
  ③ 業界トレンドとして拡大するか？

負のギャップの場合：
  ① その要因は構造的か、一時的か？
  ② 改善の可能性は？
  ③ 競合はどのようにして達成しているか？
```

#### ステップ3: 戦略的含意の抽出

```
強み領域：
  - 強みを維持・拡大する戦略
  - 他の経営指標への波及効果の確認
  - 持続性の強化

弱み領域：
  - 改善に必要なリソースと期間の評価
  - 優先順位の設定
  - ベストプラクティスの学習
```

### 複合指標による総合評価

#### マトリックス分析

複数の指標を組み合わせて評価します：

```
           高い成長率
              ↑
              |
低採算 ←─────┼─────→ 高採算
              |
              ↓
           低い成長率

・右上：理想的（成長+高採算）
・左上：投資段階（成長投資による採算圧迫）
・右下：成熟・衰退（採算は良いが成長鈍化）
・左下：問題企業（成長も採算も悪い）
```

#### 総合スコアの算出

```
正規化されたスコア = Σ（正規化指標値 × ウエイト）

ウエイト例：
  - ROE: 30% （収益性）
  - 売上成長率: 25% （成長性）
  - 流動比率: 20% （安定性）
  - 営業利益率: 15% （効率性）
  - 負債比率: 10% （財務健全性）
```

### 業種別・時間軸別の注釈

#### 業種による標準値の違い

```
IT・ソフトウェア業：
  - ROE が高い（資本効率が高い）
  - 負債比率が低い（軽資産型）

重工業・製造業：
  - ROE が相対的に低い（資本集約的）
  - 総資産回転率が低い（資産が重い）

サービス業：
  - 営業利益率が高い（原価が低い）
  - 流動比率が高い（現金商売）
```

#### 時間軸による解釈の変化

```
短期（1年）：
  - YoY の変動は正常な範囲
  - 異常値を過度に反応しない

中期（3-5年）：
  - トレンドの形成を確認
  - 構造的な優劣を判定

長期（5年以上）：
  - 業界内の序列が固定化したかを確認
  - 持続的な競争優位性の有無を判定
```

## まとめ

比較財務分析は、複数の分析手法を組み合わせることで初めて有効です：

1. **コモンサイズ分析** → 企業規模を調整した相対比較
2. **ピアグループ選定** → 公平な比較環境の構築
3. **成長率分析** → 短期と中長期のトレンド把握
4. **ギャップ分析** → 競争優位性と改善領域の特定

これらを統合することで、企業の相対的な立場、競争力、今後の成長可能性を多角的に評価できます。
