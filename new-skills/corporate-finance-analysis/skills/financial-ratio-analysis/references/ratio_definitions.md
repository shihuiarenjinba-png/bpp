# 財務比率定義書

## 概要

このドキュメントは、財務比率分析スクリプトで使用される全ての財務比率の定義、計算式、解釈ガイドを提供します。

---

## 1. 収益性指標（Profitability Ratios）

企業の利益創出能力を評価する指標です。

### 1.1 ROE（自己資本利益率 / Return on Equity）

**定義**: 株主資本に対する当期純利益の比率。株主投資に対する利益生成効率を示します。

**計算式**:
```
ROE = 当期純利益 / 自己資本
```

**業界ベンチマーク**:
- A: 15% 以上
- B: 10% ～ 15%
- C: 5% ～ 10%
- D: 0% ～ 5%
- E: 0% 未満

**解釈**:
- ROE が高い = 株主資本に対して効率的に利益を生み出している
- 一般的に 15% 以上が良好
- ただし、過度なレバレッジで達成されたROEは注意が必要
- 業種による差が大きいため、業界平均値との比較が重要

**改善方法**:
- 純利益の増加（売上増加、費用削減）
- 自己資本の削減（配当実施、自社株買収）
- ただし、過度なレバレッジは避けるべき

---

### 1.2 ROA（総資産利益率 / Return on Assets）

**定義**: 企業の全資産に対する当期純利益の比率。資産全体の利用効率を示します。

**計算式**:
```
ROA = 当期純利益 / 総資産
```

**業界ベンチマーク**:
- A: 8% 以上
- B: 5% ～ 8%
- C: 2% ～ 5%
- D: 0% ～ 2%
- E: 0% 未満

**解釈**:
- ROA が高い = 資産を効率的に活用して利益を生み出している
- ROE との差が大きい = 高いレバレッジを使用している可能性
- 業種による差が大きい（小売業は低め、金融業は高め）

**改善方法**:
- 総資産回転率の向上（売上の増加、資産効率化）
- 利益率の向上（コスト削減、価格戦略）

---

### 1.3 売上高営業利益率（Operating Profit Margin）

**定義**: 売上高に対する営業利益の比率。本業の収益性を示します。

**計算式**:
```
売上高営業利益率 = 営業利益 / 売上高
```

**業界ベンチマーク**:
- A: 15% 以上
- B: 10% ～ 15%
- C: 5% ～ 10%
- D: 0% ～ 5%
- E: 0% 未満

**解釈**:
- 本業の競争力と効率性を示す重要な指標
- 金融・特別利益を除いた純粋な事業利益性
- 業種別ばらつきが大きい

**改善方法**:
- 売上の増加（営業活動の強化）
- 売上原価の削減（生産効率化、仕入れ交渉）
- 営業費用の削減

---

### 1.4 売上高純利益率（Net Profit Margin）

**定義**: 売上高に対する当期純利益の比率。全体的な利益性を示します。

**計算式**:
```
売上高純利益率 = 当期純利益 / 売上高
```

**業界ベンチマーク**:
- A: 10% 以上
- B: 5% ～ 10%
- C: 2% ～ 5%
- D: 0% ～ 2%
- E: 0% 未満

**解釈**:
- 売上全体からどれだけ純利益を得ているかを示す
- 営業利益率より低いのが通常（税金や利息の影響）
- 企業の最終的な収益性を評価

---

### 1.5 EPS（1株当たり利益 / Earnings Per Share）

**定義**: 当期純利益を発行済み株式数で割った値。1株当たりの利益額。

**計算式**:
```
EPS = 当期純利益 / 発行済み株式数
```

**解釈**:
- 株主価値評価の基本指標
- 株価との組み合わせでPERを計算
- 増加傾向 = ポジティブな信号
- 自社株買収により分母が減少するとEPSは増加

---

### 1.6 EBITDA Margin（EBITDA利益率）

**定義**: 売上高に対するEBITDA（利息税金減価償却前利益）の比率。

**計算式**:
```
EBITDA Margin = EBITDA / 売上高
```

**業界ベンチマーク**:
- A: 20% 以上
- B: 15% ～ 20%
- C: 10% ～ 15%
- D: 5% ～ 10%
- E: 0% ～ 5%

**解釈**:
- キャッシュフロー生成能力を示す
- 減価償却政策に左右されない純粋な営業効率
- M&Aの際によく参照される指標

---

## 2. 安全性指標（Solvency Ratios）

企業の長期的な財務安定性と返済能力を評価する指標です。

### 2.1 流動比率（Current Ratio）

**定義**: 流動資産を流動負債で割った比率。短期債務返済能力を示します。

**計算式**:
```
流動比率 = 流動資産 / 流動負債
```

**業界ベンチマーク**:
- A: 1.5 以上
- B: 1.2 ～ 1.5
- C: 1.0 ～ 1.2
- D: 0.8 ～ 1.0
- E: 0.8 未満

**解釈**:
- 1.0 以上 = 短期的に返済能力がある
- 1.5 以上が安全とされる
- 過度に高い（3.0以上）= 流動資産の効率が悪い
- 1.0 未満 = 返済不能のリスク

**改善方法**:
- 流動資産の増加（現金・売上債権の増加）
- 流動負債の削減（借金返済）

---

### 2.2 当座比率（Quick Ratio）

**定義**: 流動資産から在庫を除いた金額を流動負債で割った比率。より厳密な短期支払能力。

**計算式**:
```
当座比率 = (流動資産 - 在庫) / 流動負債
```

**業界ベンチマーク**:
- A: 1.0 以上
- B: 0.8 ～ 1.0
- C: 0.6 ～ 0.8
- D: 0.4 ～ 0.6
- E: 0.4 未満

**解釈**:
- 在庫が売却できない場合を想定した、より保守的な指標
- 金融機関が重視する指標
- 最低でも 1.0 が望ましい

---

### 2.3 自己資本比率（Equity Ratio）

**定義**: 総資産に占める自己資本の割合。長期的な財務安定性を示します。

**計算式**:
```
自己資本比率 = 自己資本 / 総資産
```

**業界ベンチマーク**:
- A: 50% 以上
- B: 40% ～ 50%
- C: 30% ～ 40%
- D: 20% ～ 30%
- E: 20% 未満

**解釈**:
- 高い = 借金が少なく、経営の安定性が高い
- 40% 以上が安全とされる
- 業種による差が大きい（製造業は高め、金融業は低め）
- 低い = 倒産リスクが高い

**改善方法**:
- 利益の内部留保（配当削減）
- 増資による自己資本の増加
- 借入金の返済

---

### 2.4 D/E レシオ（Debt-to-Equity Ratio）

**定義**: 負債を自己資本で割った比率。資本構成のバランスを示します。

**計算式**:
```
D/E レシオ = 負債 / 自己資本
```

**業界ベンチマーク**:
- A: 0.0 ～ 1.0
- B: 1.0 ～ 1.5
- C: 1.5 ～ 2.0
- D: 2.0 ～ 3.0
- E: 3.0 以上

**解釈**:
- 1.0 未満 = 保守的な資本構成
- 1.0 ～ 2.0 = バランスの取れた資本構成
- 2.0 以上 = 借入比率が高い（リスク増大）
- 企業の生命線ともいえる指標

---

### 2.5 インタレストカバレッジレシオ（Interest Coverage Ratio）

**定義**: 営業利益を支払利息で割った比率。利息支払能力を示します。

**計算式**:
```
インタレストカバレッジレシオ = 営業利益 / 支払利息
```

**業界ベンチマーク**:
- A: 5.0 倍以上
- B: 3.0 ～ 5.0 倍
- C: 1.5 ～ 3.0 倍
- D: 1.0 ～ 1.5 倍
- E: 1.0 倍未満

**解釈**:
- 5.0 倍以上 = 十分な利息支払能力
- 1.5 倍未満 = 利息支払が経営を圧迫している可能性
- 金融機関が貸出判断時に重視する指標

---

## 3. 効率性指標（Efficiency Ratios）

企業の資産活用効率と運営効率を評価する指標です。

### 3.1 総資産回転率（Asset Turnover Ratio）

**定義**: 売上高を総資産で割った比率。資産全体の活用効率を示します。

**計算式**:
```
総資産回転率 = 売上高 / 総資産
```

**業界ベンチマーク**:
- A: 1.5 倍以上
- B: 1.0 ～ 1.5 倍
- C: 0.7 ～ 1.0 倍
- D: 0.4 ～ 0.7 倍
- E: 0.4 倍未満

**解釈**:
- 高い = 資産を効率的に活用している
- 業種による差が大きい（流通業は高め、不動産業は低め）
- 1.0 倍 = 総資産額と同額の売上を生成

**改善方法**:
- 売上の増加（営業強化）
- 不採算資産の売却
- 遊休資産の有効活用

---

### 3.2 棚卸資産回転率（Inventory Turnover Ratio）

**定義**: 売上高を在庫で割った比率。在庫の回転効率を示します。

**計算式**:
```
棚卸資産回転率 = 売上高 / 在庫
```

**業界ベンチマーク**:
- A: 8.0 倍以上
- B: 5.0 ～ 8.0 倍
- C: 3.0 ～ 5.0 倍
- D: 1.5 ～ 3.0 倍
- E: 1.5 倍未満

**解釈**:
- 高い = 在庫を素早く販売している
- 低い = 不良在庫や過剰在庫の可能性
- 業種による差が非常に大きい

**改善方法**:
- 在庫管理の最適化
- 販売効率の向上
- 不良在庫の処分

---

### 3.3 売上債権回転日数（Days Sales Outstanding / DSO）

**定義**: 売上債権が売上高の何日分に相当するかを示す指標。

**計算式**:
```
売上債権回転日数 = (売上債権 / 売上高) × 365 日
```

**業界ベンチマーク**:
- A: 30日以内
- B: 30 ～ 45日
- C: 45 ～ 60日
- D: 60 ～ 90日
- E: 90日以上

**解釈**:
- 低い = 売掛金を迅速に回収している
- 高い = 売掛金の回収が遅い（キャッシュフロー悪化）
- 業界慣行に左右される

**改善方法**:
- 与信管理の強化
- 回収督促の実施
- 支払条件の改善交渉

---

### 3.4 買入債務回転日数（Days Payable Outstanding / DPO）

**定義**: 買入債務が売上原価の何日分に相当するかを示す指標。

**計算式**:
```
買入債務回転日数 = (買入債務 / (売上原価)) × 365 日
```

**業界ベンチマーク**:
- 一般的に 30 ～ 60日が多い業種が多い

**解釈**:
- 高い = 支払を遅延させている（キャッシュフロー改善）
- ただし、過度に高い = 仕入先との関係悪化の可能性

**改善方法**:
- 仕入先との支払条件交渉
- 購買力の強化

---

### 3.5 CCC（キャッシュコンバージョンサイクル / Cash Conversion Cycle）

**定義**: 現金が事業活動を通じて循環する期間を示す指標。

**計算式**:
```
CCC = 売上債権回転日数 + 棚卸資産回転日数 - 買入債務回転日数
```

**業界ベンチマーク**:
- A: 負数（～ 0日）
- B: 0 ～ 30日
- C: 30 ～ 60日
- D: 60 ～ 90日
- E: 90日以上

**解釈**:
- 短い（負数） = 運転資金効率が高い
- 長い = 大量の運転資金が必要
- キャッシュフロー経営の成否を左右する重要指標

**改善方法**:
- 売上債権の早期回収
- 在庫の削減
- 支払期間の延長

---

## 4. 市場評価指標（Market Valuation Ratios）

株式市場における企業価値評価を示す指標です。

### 4.1 PER（株価収益率 / Price-to-Earnings Ratio）

**定義**: 株価を1株当たり利益（EPS）で割った比率。株価がどの程度割高・割安かを示します。

**計算式**:
```
PER = 株価 / EPS
```

**業界ベンチマーク**:
- A: 10 ～ 15倍
- B: 15 ～ 20倍
- C: 20 ～ 25倍
- D: 25 ～ 35倍
- E: 35倍以上

**解釈**:
- 低い（10倍程度） = 割安（もしくは成長期待が低い）
- 高い（25倍以上） = 割高（もしくは高成長期待）
- 業界平均値との比較が重要
- 日本平均 = 約18倍

**改善方法**:
- 利益の増加
- 自社株買収（分母の削減）

---

### 4.2 PBR（株価純資産倍率 / Price-to-Book Ratio）

**定義**: 株価を1株当たり純資産で割った比率。簿価に対する株価の倍率。

**計算式**:
```
PBR = 株価 / (自己資本 / 発行済み株式数)
```

**業界ベンチマーク**:
- A: 0.5 ～ 1.5倍
- B: 1.5 ～ 2.0倍
- C: 2.0 ～ 2.5倍
- D: 2.5 ～ 3.5倍
- E: 3.5倍以上

**解釈**:
- 1.0倍未満 = 割安（清算価値より安い）
- 1.0倍以上 = 市場が企業に価値を認めている
- ROEと組み合わせて分析（PBR ÷ ROE = 成長期待値の指標）

---

### 4.3 EV/EBITDA（企業価値倍率）

**定義**: 企業価値（時価総額+負債）をEBITDAで割った比率。業種や規模を超えた企業価値比較に用いられます。

**計算式**:
```
EV/EBITDA = (株価 × 発行済み株式数 + 負債) / EBITDA
```

**業界ベンチマーク**:
- A: 5.0 ～ 12.0倍
- B: 12.0 ～ 15.0倍
- C: 15.0 ～ 18.0倍
- D: 18.0 ～ 25.0倍
- E: 25.0倍以上

**解釈**:
- M&A価格決定の基準となりやすい
- PERより安定的な指標（税金や減価償却の影響を受けない）

---

### 4.4 配当利回り（Dividend Yield）

**定義**: 1株当たり配当金を株価で割った比率。株価に対する配当収入を示します。

**計算式**:
```
配当利回り = 1株当たり配当金 / 株価
```

**業界ベンチマーク**:
- A: 2.0% ～ 8.0%
- B: 0.8% ～ 2.0%
- C: 0.3% ～ 0.8%
- D: 0% ～ 0.3%
- E: 0% または負値

**解釈**:
- 高い = 配当が多い（価値株の特徴）
- 低い = 成長への再投資（成長株の特徴）
- 日本平均 = 約1.5% ～ 2.0%

---

### 4.5 PSR（株価売上倍率 / Price-to-Sales Ratio）

**定義**: 株価を1株当たり売上で割った比率。売上に基づく企業評価。

**計算式**:
```
PSR = 株価 / (売上高 / 発行済み株式数)
```

**業界ベンチマーク**:
- A: 0.5 ～ 2.0倍
- B: 2.0 ～ 3.0倍
- C: 3.0 ～ 4.0倍
- D: 4.0 ～ 6.0倍
- E: 6.0倍以上

**解釈**:
- PER より操作されにくい指標
- 赤字企業の評価に用いられる
- 業種による差が大きい

---

## 5. デュポン分解（DuPont Analysis）

### 5.1 3段階分解

**目的**: ROEを要因に分解し、利益性向上のポイントを特定します。

**計算式**:
```
ROE = 純利益率 × 総資産回転率 × 財務レバレッジ
    = (純利益 / 売上高) × (売上高 / 総資産) × (総資産 / 自己資本)
```

**各要素の解釈**:

1. **純利益率 = 売上原価率、営業効率**
   - 経営の収益性を示す
   - 改善：価格引き上げ、コスト削減

2. **総資産回転率 = 資産効率**
   - 資産をどの程度活用しているかを示す
   - 改善：売上増加、資産削減

3. **財務レバレッジ = 借入政策**
   - どの程度借金を活用しているかを示す
   - 改善：自己資本比率を調整

**分析例**:
- ROE が低い場合、3つの要素のどれが問題かを特定
- 各要素の業界平均との比較で改善方向を決定

### 5.2 5段階分解

**目的**: ROEをさらに詳細に分解し、根本原因を特定します。

**計算式**:
```
ROE = 営業利益率 × 純利益率 × 総資産回転率 × 財務レバレッジ
    = (営業利益 / 売上高) × (純利益 / 営業利益) × (売上高 / 総資産) × (総資産 / 自己資本)
```

**各要素の解釈**:

1. **営業利益率 = 本業の競争力**
   - 商品・サービスの競争力を示す

2. **純利益率 = 財務効率**
   - 金融費用、税率を含めた全体効率

3. **総資産回転率 = 資産効率**
   - 3段階分解と同じ

4. **財務レバレッジ = 負債活用度**
   - 3段階分解と同じ

**分析例**:
- より詳細な要因分析が可能
- 営業利益率と純利益率の差 = 金融費用や税金の影響を可視化

## 6. 比率分析のベストプラクティス

### 6.1 分析時のポイント

1. **単独指標ではなく複合的に分析**
   - 各指標は相互に関連している
   - 矛盾する指標がないか確認

2. **業界平均・競合との比較**
   - 絶対値ではなく相対的な立場を評価
   - 業界特性を考慮

3. **時系列分析（トレンド）**
   - 改善しているか、悪化しているかを確認
   - 短期的な変動と長期的なトレンドを区別

4. **経営政策との整合性確認**
   - なぜその指標値になっているのか原因分析
   - 経営方針との整合性を確認

### 6.2 指標の優先度

**最優先**:
- ROE / ROA : 企業の本質的な収益性
- 流動比率 / 自己資本比率 : 経営の安全性
- 総資産回転率 : 資産活用効率

**重要**:
- 売上高営業利益率 : 本業の競争力
- D/Eレシオ : 資本構成
- CCC : キャッシュフロー効率

**参考**:
- PER / PBR : 株価との関係
- 市場評価指標 : 投資家の期待

### 6.3 警戒信号

以下の場合は詳細分析が必要:

1. ROE が高いのに現金が少ない
   → 利益の質を確認（会計上の利益とキャッシュの乖離）

2. 売上は増加しているのに利益が減少
   → 利益率の悪化を調査

3. D/Eレシオが急上昇
   → 大型買収や新規借入がないか確認

4. CCC が拡大傾向
   → 売上債権の回収遅延や過剰在庫の可能性

5. 現金が増えているのに ROE が低下
   → 現金の有効活用ができていない可能性
