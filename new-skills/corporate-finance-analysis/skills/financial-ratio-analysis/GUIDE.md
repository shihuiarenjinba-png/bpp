# 財務比率分析スキル

## スキル概要

このスキルは、企業の財務諸表データを入力として受け取り、以下の機能を提供する包括的な財務比率分析ツールです：

### 主要機能

1. **4カテゴリの財務比率計算**
   - 収益性指標（ROE、ROA、営業利益率、純利益率、EPS、EBITDA margin）
   - 安全性指標（流動比率、当座比率、自己資本比率、D/Eレシオ、インタレストカバレッジレシオ）
   - 効率性指標（総資産回転率、棚卸資産回転率、売上債権回転日数、CCC）
   - 市場評価指標（PER、PBR、EV/EBITDA、配当利回り、PSR）

2. **デュポン分解**
   - 3段階分解（ROE = 純利益率 × 総資産回転率 × 財務レバレッジ）
   - 5段階分解（ROE のさらに詳細な分解）

3. **業界ベンチマーク比較**
   - 各比率を業界標準値と比較
   - A/B/C/D/E の評級システム

4. **可視化**
   - レーダーチャートによる総合評価
   - トレンドチャート

5. **Excel出力**
   - 複数シートによる詳細な結果報告
   - 日本語での説明とコメント

## 使用方法

### 入力データ形式

CSV または XLSX 形式の財務諸表データを準備してください。

**必須カラム：**
- 売上高 (Revenue)
- 営業利益 (Operating Income)
- 純利益 (Net Income)
- 総資産 (Total Assets)
- 流動資産 (Current Assets)
- 流動負債 (Current Liabilities)
- 在庫 (Inventory)
- 売上債権 (Accounts Receivable)
- 買入債務 (Accounts Payable)
- 自己資本 (Shareholders' Equity)
- 負債 (Total Liabilities)
- 株式数 (Shares Outstanding)
- 株価 (Stock Price) ※ 市場評価指標計算時のみ
- 配当金 (Dividends) ※ 配当利回り計算時のみ
- EBITDA ※ 任意

### 実行手順

#### 1. スクリプト実行

```bash
python scripts/ratio_calculator.py --input <入力ファイルパス> --output <出力ファイルパス>
```

**パラメータ：**
- `--input`: 入力ファイル（CSV/XLSX）のパス（必須）
- `--output`: 出力ファイル（Excel）のパス（オプション、デフォルト: `financial_analysis_report.xlsx`）

#### 2. スクリプト出力

スクリプトは以下のシートを含む Excel ファイルを生成します：

| シート名 | 内容 |
|---------|------|
| サマリー | 主要指標の総合評価、デュポン分解図 |
| 収益性指標 | ROE、ROA、営業利益率、純利益率、EPS、EBITDA margin |
| 安全性指標 | 流動比率、当座比率、自己資本比率、D/Eレシオ、インタレストカバレッジレシオ |
| 効率性指標 | 総資産回転率、棚卸資産回転率、売上債権回転日数、CCC |
| 市場評価指標 | PER、PBR、EV/EBITDA、配当利回り、PSR |
| デュポン分解 | 3段階・5段階分解の詳細 |

### 出力ファイル仕様

Excel ファイルには以下の要素が含まれます：

- **評級結果**: 各指標の A/B/C/D/E 評級
- **業界ベンチマーク比較**: 標準範囲との比較表示
- **レーダーチャート**: 主要指標の視覚的評価
- **トレンドチャート**: 時系列での推移表示（複数年データの場合）
- **説明コメント**: 各指標の解釈と改善提案

## トラブルシューティング

### エラー: "必須カラムが見つかりません"
→ 入力ファイルに必須カラムが含まれているか確認してください。カラム名は正確に一致する必要があります。

### エラー: "数値変換に失敗しました"
→ 数値カラムにテキストや空白セルが含まれていないか確認してください。

### 結果が不正常に見える場合
→ 入力データの単位（万円、百万円など）が統一されているか確認してください。
